## 0. 文字与发音

本章解决三件事：**看见假名能读准**、**听见单词能写对长短音**、**知道同一个汉字词为什么有不同读法**。日语发音不靠重音强弱区分词义，而靠`拍（モーラ）`、长短音、促音、拨音和高低音调来保持清楚。

### 0.1 假名体系与书写单位

日语书写通常混合使用汉字、平假名、片假名、数字和罗马字。语法学习时不要把罗马音当成正式文字；罗马音只适合入门标音、输入法和临时说明。

| 文字系统 | 日语名称 | 主要用途 | 例子 | 速查要点 |
| -------- | -------- | -------- | ---- | -------- |
| 平假名 | ひらがな | 助词、助动词、活用词尾、和语词、儿童读物 | は、を、ない、食べる | 语法功能主要靠平假名呈现，必须优先熟练 |
| 片假名 | カタカナ | 外来语、拟声拟态词、强调、专名 | コーヒー、メール、ドキドキ | 长音常用 `ー`，外来音会用小假名扩展 |
| 汉字 | 漢字 | 词根、实词、专有名词 | 日本、学校、読む | 同一汉字可能有音读、训读、特殊读法 |
| 罗马字 | ローマ字 | 标音、输入、护照/路牌等 | Tokyo, sushi | 学习时要回到假名，不要用英语发音读罗马字 |
| 阿拉伯数字 | 数字 | 日期、金额、编号、数量 | 2026年、3人、500円 | 实际读法取决于量词和音变 |

**最小读音单位是“拍”，不是汉语或英语意义上的音节。** 普通假名通常占 1 拍，长音、促音、拨音也各占 1 拍。

| 写法 | 读法 | 拍数 | 说明 |
| ---- | ---- | ---- | ---- |
| か | ka | 1 | 普通假名 1 拍 |
| かあ / カー | kaa | 2 | 长音多 1 拍 |
| かった | ka-t-ta | 3 | 小 `っ` 单独占 1 拍 |
| かん | ka-n | 2 | `ん` 单独占 1 拍 |
| きゃ | kya | 1 | 拗音整体 1 拍，不是 2 拍 |
| きゃあ | kyaa | 2 | 拗音 + 长音 |

**写法判断**：
  - `ゃ、ゅ、ょ、ぁ、ぃ、ぅ、ぇ、ぉ、っ` 是小假名，小假名会改变前一个音或表示停顿。
  - `や、ゆ、よ、あ、い、う、え、お、つ` 是正常大小，单独占 1 拍。
  - 横写时小假名容易看错；查词时要区分 `きや`（ki-ya，2 拍）和 `きゃ`（kya，1 拍）。

### 0.2 五十音图（平假名、片假名、罗马音）

五十音按`行`和`段`排列。`行`大致表示辅音系列，`段`表示元音。动词活用、音变和查字典都会用到行段概念。

| 行\段    | あ段（a）    | い段（i）     | う段（u）     | え段（e）    | お段（o）    |
| -------- | ------------ | ------------- | ------------- | ------------ | ------------ |
| **あ行** | あ (ア) `a`  | い (イ) `i`   | う (ウ) `u`   | え (エ) `e`  | お (オ) `o`  |
| **か行** | か (カ) `ka` | き (キ) `ki`  | く (ク) `ku`  | け (ケ) `ke` | こ (コ) `ko` |
| **さ行** | さ (サ) `sa` | し (シ) `shi` | す (ス) `su`  | せ (セ) `se` | そ (ソ) `so` |
| **た行** | た (タ) `ta` | ち (チ) `chi` | つ (ツ) `tsu` | て (テ) `te` | と (ト) `to` |
| **な行** | な (ナ) `na` | に (ニ) `ni`  | ぬ (ヌ) `nu`  | ね (ネ) `ne` | の (ノ) `no` |
| **は行** | は (ハ) `ha` | ひ (ヒ) `hi`  | ふ (フ) `fu`  | へ (ヘ) `he` | ほ (ホ) `ho` |
| **ま行** | ま (マ) `ma` | み (ミ) `mi`  | む (ム) `mu`  | め (メ) `me` | も (モ) `mo` |
| **や行** | や (ヤ) `ya` | -             | ゆ (ユ) `yu`  | -            | よ (ヨ) `yo` |
| **ら行** | ら (ラ) `ra` | り (リ) `ri`  | る (ル) `ru`  | れ (レ) `re` | ろ (ロ) `ro` |
| **わ行** | わ (ワ) `wa` | -             | -             | -            | を (ヲ) `wo` |
| **ん行** | -            | -             | ん (ン) `n`   | -            | -            |

| 项目 | 规则 | 例子 | 注意 |
| ---- | ---- | ---- | ---- |
| 元音 | あいうえお 接近 `a i u e o` | あめ、いま、うえ | `う` 不要读成汉语“乌”的圆唇感过重 |
| `し` | 罗马音常写 `shi` | しごと、しろ | 不是 `si` 的直读 |
| `ち` | 罗马音常写 `chi` | ちかい、ちず | 不是 `ti` 的直读 |
| `つ` | 罗马音常写 `tsu` | つくえ、つぎ | 不是 `tu` 的直读 |
| `ふ` | 罗马音常写 `fu` | ふゆ、ふく | 双唇轻擦，不等同英语 `f` |
| `らりるれろ` | 介于 `r/l` 之间的轻弹音 | さくら、べんり | 不要读成英语卷舌 `r` |
| `を` | 作为助词通常读 `お` | 本を読む | 罗马字可写 `wo`，实际现代读音多为 `o` |
| `は` | 作助词读 `わ` | 私は学生です | 单词内部仍读 `ha`，如 はし |
| `へ` | 作助词读 `え` | 東京へ行く | 单词内部仍读 `he`，如 へや |

**查字典和活用时的行段用途**：
  - 五段动词的变化要看词尾所属的行：書**く** → 書**か**ない、書**き**ます、書**け**ば。
  - 形容词和助动词接续常用“い形”“う段”“え段”等术语。
  - 背表不是目的，能用行段判断变化才是目的。

### 0.3 浊音、半浊音

浊音和半浊音会改变清音的发音，使得**发音更重、更有力**。浊音标记为 `゛`，半浊音标记为 `゜`。

| 行\段    | が段（ga）   | ぎ段（gi）   | ぐ段（gu）   | げ段（ge）   | ご段（go）   |
| -------- | ------------ | ------------ | ------------ | ------------ | ------------ |
| **が行** | が (ガ) `ga` | ぎ (ギ) `gi` | ぐ (グ) `gu` | げ (ゲ) `ge` | ご (ゴ) `go` |

| 行\段    | ざ段（za）   | じ段（ji）   | ず段（zu）   | ぜ段（ze）   | ぞ段（zo）   |
| -------- | ------------ | ------------ | ------------ | ------------ | ------------ |
| **ざ行** | ざ (ザ) `za` | じ (ジ) `ji` | ず (ズ) `zu` | ぜ (ゼ) `ze` | ぞ (ゾ) `zo` |

| 行\段    | だ段（da）   | ぢ段（ji/di）   | づ段（zu/du）   | で段（de）   | ど段（do）   |
| -------- | ------------ | --------------- | --------------- | ------------ | ------------ |
| **だ行** | だ (ダ) `da` | ぢ (ヂ) `ji/di` | づ (ヅ) `zu/du` | で (デ) `de` | ど (ド) `do` |

| 行\段    | ば段（ba）   | び段（bi）   | ぶ段（bu）   | べ段（be）   | ぼ段（bo）   |
| -------- | ------------ | ------------ | ------------ | ------------ | ------------ |
| **ば行** | ば (バ) `ba` | び (ビ) `bi` | ぶ (ブ) `bu` | べ (ベ) `be` | ぼ (ボ) `bo` |

| 行\段    | ぱ段（pa）   | ぴ段（pi）   | ぷ段（pu）   | ぺ段（pe）   | ぽ段（po）   |
| -------- | ------------ | ------------ | ------------ | ------------ | ------------ |
| **ぱ行** | ぱ (パ) `pa` | ぴ (ピ) `pi` | ぷ (プ) `pu` | ぺ (ペ) `pe` | ぽ (ポ) `po` |

| 现象 | 说明 | 例子 | 实用判断 |
| ---- | ---- | ---- | -------- |
| 浊音 | 清音加 `゛` | か → が、さ → ざ | 常影响词义，不能省略 |
| 半浊音 | は行加 `゜` | は → ぱ、ほ → ぽ | 只发生在は行 |
| 连浊 | 复合词后半部分开头清音变浊 | 手 + 紙 → 手紙（てがみ） | 常见但不绝对，遇到复合词要查读音 |
| `じ` / `ぢ` | 现代标准语读音多接近 | じかん、ちぢむ | 普通词多写 `じ`，重复/连浊保留 `ぢ` |
| `ず` / `づ` | 现代标准语读音多接近 | ずっと、つづく | 普通词多写 `ず`，重复/连浊保留 `づ` |
| 鼻浊音 | 部分说话者把词中 `が` 弱化 | わたしが、大学 | 现代学习不必强求，能听辨即可 |

**容易写错的对比**：

| 假名 | 常见词 | 备注 |
| ---- | ------ | ---- |
| `じ` | 時間（じかん）、自分（じぶん） | 高频默认写法 |
| `ぢ` | 縮む（ちぢむ）、鼻血（はなぢ） | 多见于重复音或复合保留 |
| `ず` | 難しい（むずかしい）、ずっと | 高频默认写法 |
| `づ` | 続く（つづく）、手作り（てづくり） | 多见于词源保留或连浊 |

### 0.4 拗音（ようおん）

拗音由 `い段假名 + 小ゃ/ゅ/ょ` 组成，整体占 **1 拍**。读写时最重要的是区分小假名和普通假名。

| 组合       | きゃ | きゅ | きょ | しゃ | しゅ | しょ | ちゃ | ちゅ | ちょ |
| ---------- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| **罗马音** | kya  | kyu  | kyo  | sha  | shu  | sho  | cha  | chu  | cho  |

| 组合       | にゃ | にゅ | にょ | ひゃ | ひゅ | ひょ | みゃ | みゅ | みょ |
| ---------- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| **罗马音** | nya  | nyu  | nyo  | hya  | hyu  | hyo  | mya  | myu  | myo  |

| 组合       | りゃ | りゅ | りょ | ぎゃ | ぎゅ | ぎょ | じゃ | じゅ | じょ |
| ---------- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| **罗马音** | rya  | ryu  | ryo  | gya  | gyu  | gyo  | ja   | ju   | jo   |

| 组合       | びゃ | びゅ | びょ | ぴゃ | ぴゅ | ぴょ |
| ---------- | ---- | ---- | ---- | ---- | ---- | ---- |
| **罗马音** | bya  | byu  | byo  | pya  | pyu  | pyo  |

| 对比 | 拍数 | 例子 | 含义/说明 |
| ---- | ---- | ---- | --------- |
| `きゃ` | 1 | 客（きゃく） | 拗音，小 `ゃ` |
| `きや` | 2 | 木屋（きや） | 两个普通假名 |
| `じゅ` | 1 | 授業（じゅぎょう） | 小 `ゅ` |
| `じゆ` | 2 | 自由（じゆう） | `ゆ` 是普通大小 |
| `しょう` | 2 | しょう / ショー | 拗音 + 长音 |

外来语会使用扩展小假名来表示传统五十音没有的音。片假名中尤其常见。

| 外来音 | 例子 | 读法提示 |
| ------ | ---- | -------- |
| ファ、フィ、フェ、フォ | ファイル、カフェ | 接近 `fa/fi/fe/fo` |
| ティ、ディ | パーティー、ディナー | `ティ` 不等于 `チ` |
| チェ、ジェ | チェック、ジェスチャー | 常见于英语借词 |
| ウィ、ウェ、ウォ | ウィスキー、ウェブ | `ウ` + 小元音 |
| ヴァ、ヴィ、ヴ、ヴェ、ヴォ | ヴァイオリン | 实际口语中也常被读成バ行近似音 |

### 0.5 促音、长音、拨音

促音、长音、拨音都会改变词义，是听力和拼写中最容易出错的部分。它们不是“附属装饰”，而是独立占拍的读音单位。

#### 0.5.1 促音：小っ / 小ッ

促音写作小 `っ` 或小 `ッ`，表示下一拍前有短暂停顿，整体占 1 拍。罗马音常把后一个辅音双写。

| 写法 | 罗马音 | 拍数 | 例子 | 对比 |
| ---- | ------ | ---- | ---- | ---- |
| きて | kite | 2 | 来て | 没有停顿 |
| きって | kitte | 3 | 切手 / 切って | `っ` 占 1 拍 |
| さか | saka | 2 | 坂 | 普通读法 |
| さっか | sakka | 3 | 作家 | 中间有停顿 |
| サカー | sakaa | 3 | 不自然写法 | 长音不是促音 |
| サッカー | sakkaa | 4 | 足球 | 促音 + 长音 |

**使用要点**：
  - 促音后通常接 `か、さ、た、ぱ` 等行的音：学校（がっこう）、切手（きって）、いっぱい。
  - 外来语中常见：バッグ、チケット、ネット。
  - 输入法中可输入双辅音：`kitte` → きって；也可输入 `xtsu` 或 `ltsu` 得到小 `っ`。

#### 0.5.2 长音：元音拉长 1 拍

长音不是“读得慢一点”，而是多 1 拍。长短音常常区分词义。

| 写法环境 | 常见写法 | 例子 | 说明 |
| -------- | -------- | ---- | ---- |
| 平假名 あ段 | `あ` | おばあさん | あ段长音多加 あ |
| 平假名 い段 | `い` | おじいさん | い段长音多加 い |
| 平假名 う段 | `う` | 空気（くうき） | う段长音多加 う |
| 平假名 え段 | 多用 `い`，少数用 `え` | 先生（せんせい）、おねえさん | 词汇化，需按词记 |
| 平假名 お段 | 多用 `う`，少数用 `お` | 学校（がっこう）、大きい（おおきい） | 高频例外要单独记 |
| 片假名 | `ー` | コーヒー、メール、スーパー | 外来语长音主要用长音符号 |

| 短音 | 长音 | 区别 |
| ---- | ---- | ---- |
| おばさん | おばあさん | 阿姨 / 奶奶、外婆 |
| おじさん | おじいさん | 叔叔 / 爷爷、外公 |
| ここ | 高校（こうこう） | 这里 / 高中 |
| ビル | ビール | 大楼 / 啤酒 |
| 雪（ゆき） | 勇気（ゆうき） | 雪 / 勇气 |

#### 0.5.3 拨音：ん / ン

拨音 `ん/ン` 独立占 1 拍。它的实际发音会受后面音影响，但写法仍是 `ん/ン`。

| 后续环境 | 实际倾向 | 例子 | 说明 |
| -------- | -------- | ---- | ---- |
| 后接 `ま、ば、ぱ` 行 | 接近 `m` | 新聞（しんぶん）、乾杯（かんぱい） | 口腔闭合，听感像 m |
| 后接 `か、が` 行 | 接近后鼻音 | 銀行（ぎんこう）、漫画（まんが） | 不要额外加元音 |
| 后接元音或や行 | 可能有轻微鼻化/分隔 | 恋愛（れんあい）、本屋（ほんや） | `ん` 仍占 1 拍 |
| 词尾 | 鼻音收尾 | 本（ほん）、時間（じかん） | 不读成 “n + 元音” |

**拼写注意**：罗马字输入时，`ん` 后接元音或や行时常要输入 `nn`，如 `honya` 可能出成 ほにゃ，输入 `honnya` 更稳。

### 0.6 音读、训读与外来语

日语汉字读法大体分为音读和训读。实际学习中，不必先背抽象定义；更实用的判断是看词的构成方式。

| 类型 | 日语名称 | 常见位置 | 例子 | 判断要点 |
| ---- | -------- | -------- | ---- | -------- |
| 音读 | 音読み（おんよみ） | 汉字复合词、正式词、抽象词 | 学校（がっこう）、電話（でんわ） | 多个汉字连在一起时常用音读 |
| 训读 | 訓読み（くんよみ） | 单个汉字词、动词/形容词词干 | 山（やま）、読む（よむ）、高い（たかい） | 汉字后接平假名活用词尾时多用训读 |
| 重箱读法 | 重箱読み | 音读 + 训读 | 重箱（じゅうばこ） | 混合读法，按词记 |
| 汤桶读法 | 湯桶読み | 训读 + 音读 | 湯桶（ゆとう） | 混合读法，按词记 |
| 熟字训 | 熟字訓 | 整个词整体训读 | 今日（きょう）、大人（おとな） | 不能逐字拆读 |
| 当字/音译 | 当て字 | 借字表示音或义 | 寿司（すし） | 以词典读法为准 |
| 外来语 | 外来語 | 片假名词 | テレビ、ホテル、コンビニ | 读法按日语音系改造，不按原语言读 |

| 汉字 | 常见音读 | 常见训读 | 例子 |
| ---- | -------- | -------- | ---- |
| 生 | せい、しょう | いきる、うまれる、なま | 学生（がくせい）、生きる（いきる）、生ビール（なまビール） |
| 行 | こう、ぎょう | いく、おこなう | 銀行（ぎんこう）、行く（いく）、行う（おこなう） |
| 上 | じょう | うえ、あがる、のぼる | 上手（じょうず）、上（うえ）、上がる（あがる） |
| 人 | じん、にん | ひと | 日本人（にほんじん）、三人（さんにん）、人（ひと） |
| 日 | にち、じつ | ひ、か | 日曜日（にちようび）、休日（きゅうじつ）、三日（みっか） |

**实用规则**：
  - 汉字复合词优先猜音读，但不要迷信：今日（きょう）、大人（おとな）必须按词记。
  - 动词和い形容词通常是“汉字词干 + 平假名活用尾”：読**む**、高**い**。
  - 人名、地名读法不稳定，必须查：日本橋 可读 にほんばし / にっぽんばし，取决于地名。
  - 外来语以日语拍为单位：クリスマス、ストレス、アイスクリーム。不要按原英语音节压缩。

### 0.7 重音、语调与拍

标准日语的重音是**高低音调**，不是英语式强弱重音。一个词内部会有高低变化；不同重音型有时能区分词义。初学阶段最重要的是先保证拍数正确，再逐步修正高低。

| 概念 | 说明 | 例子 | 学习优先级 |
| ---- | ---- | ---- | ---------- |
| 拍 | 日语节奏单位 | にほん = 3 拍，がっこう = 4 拍 | 最高，直接影响听懂和词义 |
| 音节 | 可粗略理解为发音组合 | きゃ 是 1 拍，接近 1 个音节 | 不如“拍”实用 |
| 高低重音 | 声调高低变化 | はし 可因重音不同表示不同词 | 中级后需系统训练 |
| 句调 | 整句升降 | そうです。/ そうですか。 | 会话中影响礼貌和语气 |

| 重音型 | 日语名称 | 模式说明 | 例子 |
| ------ | -------- | -------- | ---- |
| 平板型 | 平板（へいばん） | 第 1 拍低，后面高，助词也高 | さくら、学生が |
| 头高型 | 頭高（あたまだか） | 第 1 拍高，后面降 | あめ、箸（はし）某些读法 |
| 中高型 | 中高（なかだか） | 中间某处下降 | あなた、たまご 等词依词典为准 |
| 尾高型 | 尾高（おだか） | 词内不降，接助词后下降 | 花（はな）が 等词依词典为准 |

**发音检查清单**：
  - 长音是否多读了 1 拍：おばさん / おばあさん。
  - 促音是否停了 1 拍：きて / きって。
  - 拨音是否独立成拍：かな / かんな。
  - 拗音是否当作 1 拍：きや / きゃ。
  - 助词读音是否按语法变读：は→わ、へ→え、を→お。
  - 片假名外来语是否按日语拍读：コーヒー、タクシー、コンビニ。

**推荐练习法**：
  - 用手指或节拍器按拍读：がっこう = が・っ・こ・う。
  - 先读假名，再看汉字；不要直接用中文汉字音猜日语。
  - 记单词时同时记假名和重音；至少先把长音、促音、拨音写准。
  - 听写时优先标出 `ー、っ、ん、小ゃゅょ`，这些最容易造成词义错误。

## 1. 语法总览

本章是后面所有章节的地图。日语句子最核心的机制可以压缩成四句话：**谓语放句末**，**助词标关系**，**修饰语放被修饰语前**，**通过活用和接续表达时态、否定、语气和句间关系**。

### 1.1 日语的语法特点

| 特点 | 核心含义 | 典型形式 | 实用判断 |
| ---- | -------- | -------- | -------- |
| 谓语后置 | 句子的判断、动作、状态通常在最后出现 | 私は本を読む。 | 读日语时要等到句末才知道“怎么了” |
| 助词标关系 | 名词后面的助词说明它在句中的功能 | 私**は**本**を**読む。 | 不要只靠词序判断主语、宾语、方向 |
| 话题优先 | `は` 标出谈论对象，不一定是语法主语 | 私は日本語が好きです。 | `私` 是话题，`日本語` 才是喜欢的对象 |
| 修饰前置 | 修饰语放在名词或谓语前 | 昨日買った本 | 中文里“昨天买的书”，日语也是前置 |
| 黏着结构 | 词干后接助词、助动词、词尾表达语法 | 読ま-せ-られ-なかった | 按后缀一层层拆，比整块背更有效 |
| 省略频繁 | 语境明确的主语、宾语常省略 | 行きます。 | 先从上下文补“谁去、去哪、做什么” |
| 敬体和常体并存 | 同一内容可用礼貌或普通形式表达 | 行きます / 行く | 语法正确之外，还要选合适语体 |
| 名词无单复数和冠词 | 名词本身不标 a/the、单复数 | 本を買った。 | 数量、特指与否靠上下文、数量词、助词判断 |
| 时态较简洁 | 基本是非过去和过去 | 行く / 行った | 现在和将来常共用非过去，靠时间词区分 |

日语被归类为`黏着语`，许多语法意义靠后接成分表达。看复杂谓语时，可以从左到右拆：

| 形式 | 拆解 | 含义 |
| ---- | ---- | ---- |
| 読む | 読む | 读 |
| 読まない | 読ま + ない | 不读 |
| 読ませる | 読ま + せる | 让读 |
| 読ませられる | 読ま + せ + られる | 被迫读 |
| 読ませられなかった | 読ま + せ + られ + なかった | 过去没有被迫读 / 没被要求读，依语境判断 |

**阅读顺序建议**：
  - 先找句末谓语：是名词句、形容词句，还是动词句。
  - 再看谓语前的名词和助词：`は、が、を、に、で、へ、と、から、まで`。
  - 再处理修饰语：名词前的一串内容通常都在修饰后面的名词。
  - 最后看语气和连接：`です、ます、ない、た、ので、けど、ね、よ` 等。

### 1.2 基本语序与句型总览

日语基本语序常概括为 `SOV`，但更实用的理解是：**必要信息可以提前，谓语必须收束在句末**。助词使词序有一定弹性，但自然语序仍有常见排列。

| 结构槽位 | 常见内容 | 标记 | 例子 |
| -------- | -------- | ---- | ---- |
| 话题 | 这句话谈什么 | `は` | 私**は**学生です。 |
| 主体 | 动作或状态的主体 | `が` | 雨**が**降っています。 |
| 时间 | 动作发生时间 | `に`、无助词 | 7時**に**起きます。昨日行きました。 |
| 地点 | 存在、动作、经过地点 | `に、で、を` | 東京**に**住む。学校**で**勉強する。道**を**歩く。 |
| 对象 | 动作对象、目标、接受者 | `を、に、と` | 本**を**読む。友達**に**会う。彼**と**話す。 |
| 手段/原因 | 工具、方式、材料、原因 | `で` | 電車**で**行く。風邪**で**休む。 |
| 谓语 | 动作、状态、判断 | 动词、形容词、名词+だ/です | 読む。高い。学生です。 |

**常用基本句型**：

| 句型 | 功能 | 例子 | 注意 |
| ---- | ---- | ---- | ---- |
| `A は B です` | 判断、说明身份或属性 | 私は学生です。 | `は` 提示话题，`です` 使句子礼貌 |
| `A は B ではありません` | 名词句否定 | これは本ではありません。 | 口语常用 `じゃありません` |
| `A が あります` | 无生命物存在 | 机の上に本があります。 | 存在地点常用 `に` |
| `A が います` | 人或动物存在 | 教室に先生がいます。 | 人、动物用 `いる` |
| `A は B が 好きです` | 好恶、能力、感受 | 私は日本語が好きです。 | 感受对象常用 `が` |
| `A が 形容词` | 描述状态 | この部屋が暑いです。 | `が` 常提示被描述对象 |
| `A は B を V` | 及物动作 | 私は本を読みます。 | 动作对象用 `を` |
| `A は B に V` | 方向、对象、时间点 | 友達に会います。7時に起きます。 | `に` 的功能很多，要按谓语判断 |
| `A は B で V` | 动作地点、手段 | 図書館で勉強します。 | 存在地点不用 `で`，用 `に` |
| `A は B へ/に 行く` | 移动方向 | 日本へ行きます。学校に行きます。 | `へ` 强调方向，`に` 强调到达点 |

**词序调整的实际效果**：

| 句子 | 自然含义 | 语感 |
| ---- | -------- | ---- |
| 私は昨日学校で日本語を勉強しました。 | 我昨天在学校学了日语。 | 中性、自然 |
| 昨日、私は学校で日本語を勉強しました。 | 昨天，我在学校学了日语。 | 时间被提前，话题更清楚 |
| 日本語を、私は昨日学校で勉強しました。 | 日语这东西，我昨天在学校学了。 | 宾语提前，有对比或强调感 |
| 学校で日本語を勉強しました。 | 在学校学了日语。 | 主语省略，日常常见 |

### 1.3 体言、用言与活用

日语语法常把词分成两大类：**体言**和**用言**。这个区分很实用，因为接续规则往往写成“名词接什么、动词接什么、い形容词接什么”。

| 类别 | 包含 | 是否活用 | 主要功能 | 例子 |
| ---- | ---- | -------- | -------- | ---- |
| 体言 | 名词、代词、数词等 | 不活用 | 作话题、主语、宾语、补语 | 学生、これ、三人 |
| 用言 | 动词、い形容词、な形容词 | 活用或准活用 | 作谓语、修饰语，表达时态和语气 | 行く、高い、静かだ |
| 助词 | 接在词后表示关系 | 不单独活用 | 标记语法功能 | は、が、を、に、で |
| 助动词 | 接在用言等后表达语法意义 | 活用 | 否定、过去、被动、使役、推量等 | ない、た、れる、たい |
| 副词/连体词 | 修饰谓语或名词 | 通常不活用 | 补充程度、频率、样态 | とても、この、大きな |

**三类谓语是句子的核心**：

| 谓语类型 | 普通形 | 礼貌形 | 否定 | 过去 | 例子 |
| -------- | ------ | ------ | ---- | ---- | ---- |
| 名词谓语 | 学生だ | 学生です | 学生ではない | 学生だった | 彼は学生です。 |
| い形容词谓语 | 高い | 高いです | 高くない | 高かった | この本は高いです。 |
| な形容词谓语 | 静かだ | 静かです | 静かではない | 静かだった | 部屋は静かです。 |
| 动词谓语 | 行く | 行きます | 行かない | 行った | 明日行きます。 |

**活用的作用**：

| 语法意义 | 动词例 | い形容词例 | 名词/な形容词例 |
| -------- | ------ | ---------- | ---------------- |
| 非过去肯定 | 読む | 高い | 学生だ、静かだ |
| 非过去否定 | 読まない | 高くない | 学生ではない、静かではない |
| 过去肯定 | 読んだ | 高かった | 学生だった、静かだった |
| 过去否定 | 読まなかった | 高くなかった | 学生ではなかった、静かではなかった |
| 礼貌 | 読みます | 高いです | 学生です、静かです |
| 接续 | 読んで | 高くて | 学生で、静かで |

**实用判断**：看到句末时，先判断谓语类型。谓语类型决定否定、过去、接续、敬体的做法，后面章节的活用表都围绕这个判断展开。

### 1.4 敬体、常体与语体

日语不是只有“礼貌”和“不礼貌”的区别，而是根据关系、场合和文本类型选择语体。语法大全中所有句型都应同时注意**形式**和**使用场景**。

| 语体 | 日语名称 | 典型形式 | 使用场景 | 例子 |
| ---- | -------- | -------- | -------- | ---- |
| 敬体 | 丁寧体 | です、ます | 对陌生人、老师、客户、正式说明 | 明日行きます。 |
| 常体 | 普通体 | だ、する、ない、た | 朋友、家人、日记、论文从句 | 明日行く。 |
| 书面体 | 書き言葉 | である、する、〜において | 论文、报告、新闻、正式文章 | これは重要である。 |
| 口语体 | 話し言葉 | じゃ、ちゃう、てる | 日常会话、聊天 | もう食べちゃった。 |
| 敬语 | 敬語 | なさる、いたす、ございます | 商务、服务、上下关系 | 少々お待ちください。 |

**同一内容的语体转换**：

| 功能 | 常体 | 敬体 | 更正式 |
| ---- | ---- | ---- | ------ |
| 名词句 | これは本だ。 | これは本です。 | これは本である。 |
| 动词句 | 明日行く。 | 明日行きます。 | 明日伺います。 |
| 否定 | 行かない。 | 行きません。 | 参りません。 |
| 过去 | 行った。 | 行きました。 | 伺いました。 |
| 请求 | 来て。 | 来てください。 | お越しください。 |

**语体选择原则**：
  - 对不熟的人，默认用 `です/ます`。
  - 写说明文、笔记、语法解释时，可用简洁的常体或书面体，但例句最好标出礼貌形。
  - 从句内部常用普通形，即使主句是敬体：明日行く**と思います**。
  - 敬语不是简单把句尾换成 `ます`，而是会改变动词和表达方式：見る → ご覧になる、見る → 拝見する。

### 1.5 接续规则总览

日语句型最难的地方常常不是意思，而是“前面接什么形”。所以每个语法点都应该同时记三件事：**意义**、**接续**、**使用限制**。

| 接续标签 | 含义 | 例子 | 常见后续 |
| -------- | ---- | ---- | -------- |
| 辞书形 | 动词基本形 | 行く、食べる | こと、つもり、前に、ために |
| ます形词干 | 去掉 `ます` 的形式 | 行き、食べ | たい、ながら、やすい、に行く |
| て形 | 连接、请求、状态 | 行って、食べて | ください、いる、しまう、から |
| た形 | 过去、完成 | 行った、食べた | ことがある、あとで、ら |
| ない形 | 否定基底 | 行かない、食べない | で、ければならない、ように |
| 普通形 | 非敬体句子形式 | 行く、行った、行かない | と思う、そうだ、から、ので |
| 连体形 | 修饰名词的形式 | 行く人、読んだ本 | 名词、こと、もの、ところ |
| 名词 + の | 名词接续形式 | 学生の、雨の | ために、ように、せいで、に |
| な形容词 + な | 修饰名词 | 静かな町 | 名词、こと、もの |
| な形容词 + に | 修饰谓语或表示变化结果 | 静かに話す、きれいになる | 动词、なる、する |

**常见接续模式速查**：

| 模式 | 形式 | 功能 | 例子 |
| ---- | ---- | ---- | ---- |
| 并列动作 | Vて、Vて | 顺序、并列、轻原因 | 朝起きて、顔を洗います。 |
| 时间先后 | Vてから | 做完前项再做后项 | 宿題をしてから寝ます。 |
| 名词化 | 普通形 + こと/の | 把句子变成名词成分 | 日本語を勉強することが好きです。 |
| 引用判断 | 普通形 + と思う | 认为、觉得 | 明日雨が降ると思います。 |
| 原因理由 | 普通形 + から/ので | 因为 | 忙しいので、行けません。 |
| 逆接 | 普通形 + けど/のに | 但是、明明却 | 勉強したのに、忘れました。 |
| 条件 | たら、ば、なら、と | 如果、一……就 | 雨が降ったら、行きません。 |
| 目的 | 辞书形 + ために / ように | 为了 | 合格するために勉強します。 |
| 变化 | 名词/な形容词 + に + なる | 变成 | 医者になりました。 |
| 决定 | 名词/动词辞书形 + にする | 选择、决定 | コーヒーにします。 |

**学习和查表时的三步法**：
  - 先问“这个句型表达什么功能”：原因、条件、目的、让步、推量、义务、许可等。
  - 再问“前面接什么形式”：动词辞书形、た形、ない形、名词+の、な形容词+な 等。
  - 最后问“语体和限制是什么”：口语/书面、礼貌/普通、能不能用于自己、是否带负面语气。

**一个句子的拆解示例**：

| 层次 | 片段 | 功能 |
| ---- | ---- | ---- |
| 话题 | 私は | 说明谈论对象是“我” |
| 时间 | 昨日 | 说明动作时间 |
| 地点 | 図書館で | `で` 表示动作地点 |
| 对象 | 日本語の本を | `の` 修饰，`を` 标动作对象 |
| 谓语 | 読みました | 动词 `読む` 的礼貌过去形 |

完整句：私は昨日図書館で日本語の本を読みました。  
理解顺序：先看句末 `読みました`，知道是“读了”；再看 `を` 前面是读的对象；再看 `で` 前面是动作地点；最后补出话题和时间。

## 2. 句子成分、词性与信息结构

本章解决两个核心问题：**一个词在句子里承担什么功能**，以及**一句话把什么信息当作已知、重点、新信息**。日语中词序比汉语灵活，但不是随便排；助词、修饰位置、谓语类型共同决定句子结构。

### 2.1 句子成分

句子成分是“功能”，词性是“材料”。同一个名词可以当话题、主语、宾语、补语或修饰语，关键看它后面的助词和所在位置。

| 句子成分 | 日语名称 | 主要功能 | 常见标记 | 典型位置 | 例子 |
| -------- | -------- | -------- | -------- | -------- | ---- |
| 话题 | 主題（しゅだい） | 表示这句话谈论什么 | `は、なら、って` | 句首或被提前 | 私**は**学生です。 |
| 主语 | 主語（しゅご） | 动作、状态、存在的主体 | `が`，有时 `は` | 谓语前 | 雨**が**降っています。 |
| 谓语 | 述語（じゅつご） | 句子的核心判断、动作、状态 | 动词、形容词、名词+だ/です | 句末 | 本を**読みます**。 |
| 宾语 | 目的語（もくてきご） | 动作直接作用的对象 | `を` | 谓语前 | 本**を**読みます。 |
| 补语 | 補語（ほご） | 补充方向、对象、结果、存在地点等 | `に、へ、と、から、まで` | 谓语前 | 友達**に**会う。 |
| 状语 | 連用修飾語 | 修饰谓语，说明时间、地点、方式、程度 | `で、に、から、まで`，副词 | 谓语前 | 図書館**で**静かに読む。 |
| 定语 | 連体修飾語 | 修饰名词，说明所属、性质、动作关系 | `の、な、い`，从句 | 名词前 | 昨日買った**本**。 |
| 接续语 | 接続語 | 连接句子，提示逻辑关系 | 接续词、接续助词 | 句首或分句末 | しかし、雨が降った。 |
| 独立语 | 独立語 | 呼唤、感叹、应答 | 感动词、称呼 | 句首或独立 | はい、分かりました。 |

**句子成分速判**：

| 看到的形式 | 优先判断 | 例子 | 注意 |
| ---------- | -------- | ---- | ---- |
| `名词 + は` | 话题、对比 | 私は行きます。 | 不一定是主语，也可能提示宾语或地点 |
| `名词 + が` | 主语、新信息、状态对象 | 猫がいます。日本語が好きです。 | 在能力、好恶、感受句里常标对象 |
| `名词 + を` | 宾语、经过点、离开点 | 本を読む。橋を渡る。家を出る。 | 不总是“宾语”，要看动词 |
| `名词 + に` | 到达点、存在地点、对象、时间点、结果 | 学校に行く。机にある。先生に聞く。 | 功能最多，必须按谓语判断 |
| `名词 + で` | 动作地点、手段、范围、原因 | 学校で勉強する。電車で行く。 | 存在地点一般不用 `で` |
| `名词 + の + 名词` | 所属、属性、同位、内容 | 私の本、日本語の先生 | 不要机械翻成“的” |
| `动词/形容词 + 名词` | 连体修饰 | 昨日読んだ本、高い店 | 整个小句修饰后面名词 |
| `副词 + 谓语` | 连用修饰 | とても高い、ゆっくり話す | 副词通常不直接修饰名词 |

**完整拆句示例**：

| 片段 | 成分 | 说明 |
| ---- | ---- | ---- |
| 私は | 话题 | 这句话谈“我” |
| 昨日 | 状语 | 时间 |
| 駅で | 状语 | 动作地点 |
| 友達に | 补语 | 见面的对象 |
| 新しい本を | 宾语 | `新しい` 修饰 `本`，`を` 标对象 |
| 渡しました | 谓语 | `渡す` 的礼貌过去形 |

完整句：私は昨日駅で友達に新しい本を渡しました。  
理解顺序：先看句末 `渡しました`，再找“把什么给谁、在哪里、什么时候、谁作为话题”。

### 2.2 品词体系

品词是词的语法性质。学日语时最重要的不是背名称，而是知道它能不能活用、能接什么、能不能直接作谓语。

| 词性 | 日语名称 | 功能 | 语法特征 | 常见句子成分 | 举例 |
| ---- | -------- | ---- | -------- | ------------ | ---- |
| 名词 | 名詞 | 人、事物、地点、时间、抽象概念名称 | 不活用；后接助词、だ/です、の | 话题、主语、宾语、补语、定语 | 学生、日本、時間、勉強 |
| 代词 | 代名詞 | 指代人、物、方向、场所 | 多作为名词使用 | 话题、主语、宾语 | 私、これ、そこ、どちら |
| 数词/数量词 | 数詞・助数詞 | 表示数量、顺序、频率 | 常与量词结合；读音多变 | 状语、定语、补语 | 一人、三冊、二回 |
| 动词 | 動詞 | 动作、变化、存在 | 活用；决定句子时态、极性、语气 | 谓语、定语、状语成分的一部分 | 書く、食べる、する、ある |
| い形容词 | 形容詞 | 性质、状态 | 以 `い` 结尾并活用 | 谓语、定语、状语 | 高い、寒い、面白い |
| な形容词 | 形容動詞 | 性质、状态 | 修饰名词接 `な`；作谓语接 `だ/です` | 谓语、定语、状语 | 静か、有名、便利 |
| 副词 | 副詞 | 修饰动作、状态、程度、频率 | 通常不活用；不直接作谓语 | 状语 | とても、ゆっくり、たぶん |
| 连体词 | 連体詞 | 专门修饰名词 | 后面必须接名词；不作谓语 | 定语 | この、その、大きな、ある |
| 接续词 | 接続詞 | 连接句子或段落 | 独立使用；不活用 | 接续语 | しかし、だから、そして |
| 助词 | 助詞 | 标示关系、范围、语气 | 附着在词或句子后；不独立 | 成分标记 | は、が、を、に、で、も、ね |
| 助动词 | 助動詞 | 否定、过去、被动、使役、推量、礼貌 | 接在用言等后；有活用 | 谓语的一部分 | ない、た、れる、せる、です |
| 感动词 | 感動詞 | 应答、呼唤、感叹 | 可独立成句 | 独立语 | はい、いいえ、ああ、もしもし |

**词性和接续的关系**：

| 前项类型 | 修饰名词 | 修饰谓语 | 作谓语 | 例子 |
| -------- | -------- | -------- | ------ | ---- |
| 名词 | `名词 + の + 名词` | `名词 + で/に` 等 | `名词 + だ/です` | 日本語の本、学生です |
| 动词 | 普通形直接接名词 | て形、ます形词干等接续 | 直接作谓语 | 読む本、読んでいる、読む |
| い形容词 | 直接接名词 | `く` 形修饰谓语 | 直接作谓语 | 高い本、高く売る、高い |
| な形容词 | `な + 名词` | `に + 谓语` | `だ/です` | 静かな町、静かに話す、静かだ |
| 副词 | 通常不能直接修饰名词 | 直接修饰谓语/形容词 | 通常不能作谓语 | とても高い、ゆっくり歩く |
| 连体词 | 直接接名词 | 不能修饰谓语 | 不能作谓语 | この本、大きな問題 |

**容易混淆的词类**：

| 对比 | 区别 | 例子 | 判断法 |
| ---- | ---- | ---- | ------ |
| い形容词 vs な形容词 | い形容词直接修饰名词；な形容词要加 `な` | 高い本 / 静かな本 | 看修饰名词时是否需要 `な` |
| 副词 vs 连体词 | 副词修饰谓语；连体词修饰名词 | とても高い / この本 | 后面接动词形容词多为副词，接名词可能是连体词 |
| 名词 vs な形容词 | 都可接 `です`，但な形容词能表示性质 | 学生です / 静かです | 看能否说 `静かな町` 这种 `な+名词` |
| 助词 vs 助动词 | 助词标关系；助动词改变谓语语法意义 | 本を / 読まない | 能不能参与谓语活用是关键 |
| 接续词 vs 接续助词 | 接续词独立连接句子；接续助词黏在前句后 | しかし、雨だ。/ 雨だが、行く。 | 是否附着在前一成分后 |

### 2.3 主题、主语与焦点

`は` 和 `が` 不是简单的“主语助词二选一”。粗略说：`は` 提示话题或对比，`が` 标示主语、新信息或焦点。很多自然表达不是看汉语“我/他”是不是主语，而是看说话者想把什么放进背景、把什么作为重点。

| 项目 | `は` | `が` |
| ---- | ---- | ---- |
| 基本功能 | 话题、对比、已知信息框架 | 主语、新信息、焦点、排他强调 |
| 常见语感 | “至于……、说到……” | “正是……、有……、……才是” |
| 信息状态 | 往往是双方已知或要拿来谈的对象 | 往往是新出现、被发现、被回答的对象 |
| 句子范围 | 可统领整句或后续多句 | 多局限在所在谓语或从句内 |
| 否定句 | 常用于整体话题或对比 | 常用于存在、能力、感受对象 |
| 疑问回答 | 回答“关于X怎么样” | 回答“谁/什么是X” |

| 用法 | 例子 | 说明 |
| ---- | ---- | ---- |
| 话题 | 私**は**学生です。 | 说到我，我是学生 |
| 新信息 | 昔々、おじいさん**が**いました。 | 故事中首次引入人物 |
| 疑问词主语 | 誰**が**来ましたか。 | 疑问词作主语通常用 `が` |
| 排他强调 | 私**が**やります。 | 不是别人，是我来做 |
| 对比 | 魚**は**食べますが、肉**は**食べません。 | 鱼吃，肉不吃 |
| 状态对象 | 私は日本語**が**好きです。 | 话题是我，喜欢的对象是日语 |
| 存在主体 | 机の上に本**が**あります。 | 存在的是书 |
| 从句主语 | 彼**が**来た時、私は家にいました。 | 从句内部主语多用 `が` |

**问答判断**：

| 问题 | 自然回答 | 为什么 |
| ---- | -------- | ------ |
| 誰が行きますか。 | 私**が**行きます。 | 回答“谁”，焦点在 `私` |
| 田中さんは行きますか。 | 田中さん**は**行きます。 | 话题已给定，说明田中去不去 |
| 何が好きですか。 | 日本語**が**好きです。 | 回答喜欢的对象 |
| 日本語は好きですか。 | 日本語**は**好きです。 | 把日语作为话题，可能暗含对比 |

**常见误区**：
  - `は` 不等于主语。`私は寿司が好きです` 中，`私` 是话题，`寿司` 是感受对象。
  - `が` 不一定要翻成“是……”。很多时候只是自然标记存在、状态或新信息。
  - `は` 可替换其他助词形成话题：東京**では**便利です、日曜日**には**行けます。
  - 一个句子可以同时有 `は` 和 `が`，通常是“话题 + 焦点/对象/从句主语”。

### 2.4 省略、倒装与语境

日语大量依赖语境。只要说话双方知道是谁、什么对象、什么地点，句子中就可以省略。学习时不能把每句都按汉语补成完整主谓宾，否则会显得生硬。

| 省略对象 | 例子 | 补全理解 | 使用场景 |
| -------- | ---- | -------- | -------- |
| 主语 | 行きます。 | 私/彼/我们要去。 | 对话中主体已知 |
| 宾语 | もう食べました。 | 已经吃了它/饭。 | 对象已在上下文出现 |
| 话题 | 高いですね。 | 这个东西很贵啊。 | 指着某物或接续上文 |
| 谓语一部分 | コーヒーを。 | 请给我咖啡。 | 点餐、请求、口语省略 |
| 后续结论 | ちょっと用事があって…… | 因为有点事，所以…… | 委婉拒绝、留白 |

**省略恢复步骤**：
  - 看上一句或场景：正在谈谁、看什么、做什么。
  - 看敬语方向：`いただく、くださる、差し上げる` 能提示动作双方。
  - 看谓语选择：`行く/来る/あげる/くれる` 常暗示说话者视角。
  - 不确定时不要强补中文主语，先按“有人/某物/这件事”理解。

倒装是为了强调、补充或模拟口语节奏。日语句末通常是谓语，但会话里常把补充成分放到后面。

| 正常语序 | 倒装/补充 | 语感 |
| -------- | --------- | ---- |
| 私は行きません。 | 行きません、私は。 | 强调“我不去”，可能与别人对比 |
| それは無理です。 | 無理です、それは。 | 先给判断，再补话题 |
| 昨日見ました。 | 見ました、昨日。 | 先说动作，再补时间 |
| 本当においしいですね。 | おいしいですね、本当に。 | 追加强调 |

**实用原则**：
  - 书面语和考试作文中优先使用正常语序。
  - 会话、小说、台词中倒装很多，理解时仍要回到“助词 + 谓语”的关系。
  - 省略不是语法缺失，而是日语自然表达的一部分。

### 2.5 修饰结构：连体修饰与连用修饰

日语修饰语通常放在被修饰语前面。修饰名词叫连体修饰，修饰谓语、形容词或整句叫连用修饰。掌握修饰结构，才能读懂长句。

#### 2.5.1 连体修饰：修饰名词

| 修饰类型 | 形式 | 例子 | 说明 |
| -------- | ---- | ---- | ---- |
| 名词修饰名词 | `名词 + の + 名词` | 日本語の本 | 所属、内容、属性等 |
| い形容词修饰名词 | `い形容词 + 名词` | 高い本 | 直接接名词 |
| な形容词修饰名词 | `な形容词 + な + 名词` | 静かな町 | 必须加 `な` |
| 连体词修饰名词 | `连体词 + 名词` | この本、大きな問題 | 连体词不能单独作谓语 |
| 动词修饰名词 | `动词普通形 + 名词` | 昨日買った本 | 整个动作小句修饰名词 |
| 句子修饰名词 | `普通形小句 + 名词` | 友達がくれた本 | 小句内部主语常用 `が` |

**长名词短语拆解**：

| 名词短语 | 中心名词 | 修饰部分 | 含义 |
| -------- | -------- | -------- | ---- |
| 昨日買った本 | 本 | 昨日買った | 昨天买的书 |
| 先生が紹介してくれた店 | 店 | 先生が紹介してくれた | 老师介绍给我的店 |
| 日本語を勉強している学生 | 学生 | 日本語を勉強している | 正在学日语的学生 |
| 駅の近くにある静かなカフェ | カフェ | 駅の近くにある、静かな | 车站附近的安静咖啡店 |

**注意**：日语连体修饰中，被修饰名词和修饰小句的关系不一定明说，要靠语义判断。

| 形式 | 可能关系 | 例子 |
| ---- | -------- | ---- |
| 私が読んだ本 | 本是被读的对象 | 我读过的书 |
| 私が買った店 | 店是买东西的地点，严格说应理解为“我买东西的店” | 口语可见，但写作可改成 私が買い物をした店 |
| 友達が来る日 | 日是动作发生时间 | 朋友来的日子 |
| ご飯を食べる時間 | 时间是动作发生时间 | 吃饭时间 |

#### 2.5.2 连用修饰：修饰谓语或整句

| 修饰类型 | 形式 | 例子 | 说明 |
| -------- | ---- | ---- | ---- |
| 副词修饰 | `副词 + 谓语/形容词` | ゆっくり話す、とても高い | 表示方式、程度、频率 |
| 名词+助词 | `名词 + で/に/から/まで` | 学校で勉強する、朝に起きる | 表示地点、时间、手段等 |
| い形容词连用形 | `〜く + 动词/なる` | 早く起きる、高くなる | 修饰动作或表示变化 |
| な形容词连用形 | `〜に + 动词/なる` | 静かに話す、きれいになる | 修饰动作或表示变化 |
| 动词て形 | `Vて + 后续` | 歩いて帰る、読んでいる | 顺接、状态、方式等 |
| 接续从句 | `から、ので、けど、のに` 等 | 雨が降ったので、帰った | 修饰整句逻辑关系 |

**修饰范围判断**：

| 句子 | 修饰关系 | 说明 |
| ---- | -------- | ---- |
| とても高い本を買った。 | `とても` 修饰 `高い` | 买了非常贵的书 |
| 静かに本を読んだ。 | `静かに` 修饰 `読んだ` | 安静地读书 |
| 昨日買った本を今日読んだ。 | `昨日買った` 修饰 `本`；`今日` 修饰 `読んだ` | 昨天买的书今天读了 |
| 雨が降ったので、家にいました。 | 前句说明原因 | 因为下雨，所以待在家 |

**读长句的实用步骤**：
  - 找最后的谓语，确定整句核心动作或判断。
  - 找谓语前的 `を、に、で、が、は`，确定主要成分。
  - 从每个名词往前看，收集修饰它的内容。
  - 遇到 `ので、けど、たら、ながら` 等接续形式，先切成分句。
  - 最后按中文自然顺序重组，不要逐词硬翻。

## 3. 名词系统

日语名词本身通常**不标单复数、不标性别、不标冠词，也不按时态变化**。名词要进入句子，主要靠三类东西：后接助词、后接 `だ/です` 构成谓语、用 `の` 或修饰从句连接其他名词。

### 3.1 名词谓语

名词谓语用于说明身份、类别、属性、时间、地点等。核心结构是 `名词 + だ/です`。中文常不说“是”，但日语句末通常要有谓语形式。

| 功能 | 结构 | 例子 | 说明 |
| ---- | ---- | ---- | ---- |
| 身份 | `A は B です` | 私は学生です。 | A 是 B |
| 类别 | `これは B です` | これは辞書です。 | 说明事物类别 |
| 属性 | `A は B です` | 今日は休みです。 | B 可以是状态性名词 |
| 时间 | `今は B です` | 今は三時です。 | 时间名词作谓语 |
| 地点/场合 | `場所は B です` | 会議は三階です。 | 说明地点或安排 |
| 解释 | `つまり B です` | これは間違いです。 | 给出判断或归类 |

**名词谓语活用速查**：

| 语体/时态 | 肯定 | 否定 | 例子 |
| --------- | ---- | ---- | ---- |
| 常体非过去 | 学生だ | 学生ではない / 学生じゃない | 彼は学生だ。 |
| 敬体非过去 | 学生です | 学生ではありません / 学生じゃありません | 彼は学生です。 |
| 常体过去 | 学生だった | 学生ではなかった / 学生じゃなかった | 昔、彼は学生だった。 |
| 敬体过去 | 学生でした | 学生ではありませんでした / 学生じゃありませんでした | 昔、彼は学生でした。 |
| 书面体 | 学生である | 学生ではない | 彼は学生である。 |
| 接续形 | 学生で | 学生ではなく / 学生じゃなく | 学生で、研究者でもある。 |

| 形式 | 语感 | 使用场景 |
| ---- | ---- | -------- |
| `ではない` | 标准、书面感较强 | 说明、文章、较正式会话 |
| `じゃない` | 口语、自然 | 日常会话 |
| `ではありません` | 礼貌、标准 | 正式口语、课堂、服务场景 |
| `じゃありません` | 礼貌但稍口语 | 日常礼貌会话 |
| `である` | 论文、报告、定义 | 书面说明，不用于普通会话 |

**名词句常见扩展**：

| 结构 | 功能 | 例子 | 说明 |
| ---- | ---- | ---- | ---- |
| `A は B ではなく C です` | 不是 B，而是 C | これは本ではなく、雑誌です。 | 对比修正 |
| `A は B でもあります` | 也是 B | 彼は医者でもあります。 | `も` 加在 `で` 后强调兼具身份 |
| `A は B ではありませんか` | 难道不是 B 吗 | これはあなたの本ではありませんか。 | 礼貌疑问或确认 |
| `A は B でしょう` | 大概是 B | 明日は雨でしょう。 | 推量，较礼貌 |
| `A は B かもしれません` | 可能是 B | 彼は先生かもしれません。 | 不确定判断 |
| `A は B のようです` | 像 B / 似乎是 B | これは夢のようです。 | 比况或推测 |

**名词谓语的常见错误**：

| 错误 | 正确 | 原因 |
| ---- | ---- | ---- |
| 私は学生。 | 私は学生です。/ 私は学生だ。 | 单独名词可口语省略，但学习和正式表达要有谓语形式 |
| これは私の本ありません。 | これは私の本ではありません。 | 名词否定要用 `ではない/ではありません` |
| 昨日は休みです。 | 昨日は休みでした。 | 过去时间下名词谓语要用过去形 |
| 彼は親切です人。 | 彼は親切な人です。 | な形容词修饰名词要加 `な` |

### 3.2 代词与人称表达

日语代词不像汉语“我、你、他”那样频繁使用。只要语境明确，主语和宾语常省略。直接称呼对方时，优先使用姓名、职称、称呼语，而不是 `あなた`。

| 人称 | 常见形式 | 语感/场合 | 例子 | 注意 |
| ---- | -------- | --------- | ---- | ---- |
| 第一人称 | 私（わたし） | 中性、礼貌、最安全 | 私は学生です。 | 男女都可用 |
| 第一人称 | 僕（ぼく） | 男性常用，较柔和 | 僕も行きます。 | 正式场合不如 私 安全 |
| 第一人称 | 俺（おれ） | 男性口语，随意/强势 | 俺は知らない。 | 对上级、正式场合避免 |
| 第一人称 | わたくし | 非常正式、谦逊 | わたくしが担当いたします。 | 商务、演讲 |
| 第二人称 | あなた | 中性但有距离，夫妻间也可用 | あなたはどう思いますか。 | 直接对人常显得生硬 |
| 第二人称 | 君（きみ） | 对下、亲近或文学语感 | 君はどうする？ | 对上级不能用 |
| 第二人称 | お前（おまえ） | 粗鲁或非常亲密 | お前も来い。 | 学习者慎用 |
| 第三人称 | 彼（かれ） | 他；也可指男朋友 | 彼は先生です。 | 会话中常用姓名替代 |
| 第三人称 | 彼女（かのじょ） | 她；也可指女朋友 | 彼女は医者です。 | 语境决定含义 |
| 复数 | 私たち、彼ら、彼女たち | 我们、他们、她们 | 私たちは学生です。 | 日语名词本身不标复数，必要时才加 |

**称呼优先级**：

| 场景 | 推荐称呼 | 例子 | 说明 |
| ---- | -------- | ---- | ---- |
| 同事/同学 | 姓名 + さん | 田中さんは行きますか。 | 最安全的普通称呼 |
| 老师/医生/律师等 | 姓名/职业 + 先生 | 佐藤先生、先生 | `先生` 可作称呼 |
| 公司上级 | 姓名 + 役职 | 鈴木部長、山田課長 | 头衔本身可作敬称 |
| 店员称顾客 | お客様 | お客様、こちらへどうぞ。 | 服务业固定称呼 |
| 家庭成员 | お父さん、母、兄 | 父は会社員です。 | 对外说自己家人常用谦称：父、母 |
| 不知道姓名 | すみません | すみません、これは何ですか。 | 避免直接说 `あなた` |

**指代人的常见表达**：

| 表达 | 含义 | 例子 | 注意 |
| ---- | ---- | ---- | ---- |
| この人 / その人 / あの人 | 这个人/那个人 | あの人は誰ですか。 | 中性，但当面说可能不够礼貌 |
| この方 / その方 / あの方 | 这位/那位 | あの方は先生です。 | `人` 的礼貌说法 |
| どなた | 哪位 | どなたですか。 | 比 `誰` 礼貌 |
| 誰 | 谁 | 誰が来ますか。 | 普通疑问 |
| 自分 | 自己；有时指“我” | 自分でやります。 | 关西/男性口语中可作第一人称，学习者先按“自己”理解 |

**实用原则**：
  - 能省略时就省略，不要每句都说 `私は`。
  - 不知道怎么称呼对方时，用 `すみません` 开头更自然。
  - 对上级、客户、陌生人，避免 `君、お前`。
  - `彼/彼女` 在会话中没有英语 he/she 那么高频，很多时候直接用姓名。

### 3.3 指示词：こそあど体系

`こそあど` 是日语指示词系统：`こ` 近说话人，`そ` 近听话人或前文刚提到，`あ` 离双方都远或双方都知道，`ど` 表疑问。

| 系列 | 距离/信息 | 中文近似 | 例子 |
| ---- | --------- | -------- | ---- |
| こ- | 靠近说话人；马上要说的内容 | 这 | これ、この、ここ、こちら |
| そ- | 靠近听话人；刚提到的内容 | 那/这个前文项 | それ、その、そこ、そちら |
| あ- | 远离双方；双方共同知道的内容 | 那个远处/那件事 | あれ、あの、あそこ、あちら |
| ど- | 疑问 | 哪 | どれ、どの、どこ、どちら |

**基本指示词速查**：

| 功能 | こ | そ | あ | ど | 用法 |
| ---- | -- | -- | -- | -- | ---- |
| 事物 | これ | それ | あれ | どれ | 单独作名词：これは本です |
| 修饰名词 | この | その | あの | どの | 后面必须接名词：この本 |
| 场所 | ここ | そこ | あそこ | どこ | ここで待ちます |
| 方向/人/礼貌场所 | こちら | そちら | あちら | どちら | こちらは田中さんです |
| 这样/那样 | こんな | そんな | あんな | どんな | こんな本が好きです |
| 这样地 | こう | そう | ああ | どう | こうしてください |
| 这种程度 | これくらい | それくらい | あれくらい | どれくらい | どれくらいかかりますか |

**`これ` 和 `この` 的区别**：

| 形式 | 词性/功能 | 例子 | 说明 |
| ---- | --------- | ---- | ---- |
| `これ` | 代词，可单独用 | これは本です。 | “这个”本身作名词 |
| `この` | 连体词，后接名词 | この本は高いです。 | “这个……”不能单独用 |
| `それ` | 代词 | それをください。 | “那个”作对象 |
| `その` | 连体词 | その本をください。 | “那本……” |
| `どれ` | 代词 | どれが好きですか。 | 哪一个 |
| `どの` | 连体词 | どの本が好きですか。 | 哪本 |

**空间和篇章用法**：

| 用法 | 例子 | 说明 |
| ---- | ---- | ---- |
| 现场近说话人 | これは私の本です。 | 书在说话人附近 |
| 现场近听话人 | それを見せてください。 | 东西在听话人附近 |
| 现场远离双方 | あれは何ですか。 | 东西在远处 |
| 指前文 | 昨日映画を見ました。それはとても面白かったです。 | `それ` 指刚提到的电影 |
| 指双方共同记忆 | あの店、覚えていますか。 | 说话双方都知道的那家店 |
| 指后文 | これから説明します。 | `これから` 表从现在起、接下来 |

**礼貌和方向**：

| 表达 | 含义 | 例子 | 说明 |
| ---- | ---- | ---- | ---- |
| こちら | 这边；这位；我方 | こちらは山田さんです。 | 可礼貌介绍人 |
| そちら | 那边；贵方/贵处 | そちらのご都合はいかがですか。 | 商务中可指对方 |
| あちら | 那边；那位 | あちらでお待ちください。 | 礼貌指示方向 |
| どちら | 哪边；哪位；哪里 | どちらから来ましたか。 | 比 `どこ/誰` 更礼貌 |

### 3.4 数词、量词与助数词

日语数量表达通常是 `数字 + 助数词`。助数词会随对象种类变化，且常发生读音变化。数量词在句中可放在名词前，也可放在动词前作数量状语。

| 表达位置 | 结构 | 例子 | 语感/用途 |
| -------- | ---- | ---- | --------- |
| 名词前 | `数量 + の + 名词` | 三冊の本を買いました。 | 强调“几本书”这个名词短语，书面也常见 |
| 动词前 | `名词 + を + 数量 + 动词` | 本を三冊買いました。 | 最自然的数量说明，日常高频 |
| 话题化 | `数量 + は/も` | 三人は来ます。三人も来ました。 | `は` 至少/就该数量，`も` 多达 |
| 大约 | `数量 + くらい/ぐらい` | 十分くらい待ちました。 | 约数 |
| 只 | `数量 + だけ/しか` | 一つだけあります。一つしかありません。 | `しか` 必须配否定 |

**常用助数词**：

| 助数词 | 对象 | 例子 | 注意 |
| ------ | ---- | ---- | ---- |
| 人（にん） | 人数 | 三人（さんにん） | 一人（ひとり）、二人（ふたり）特殊 |
| つ | 泛用物品、抽象事项 | 一つ、二つ | 到十常用和语读法 |
| 個（こ） | 小物、具体物 | りんごを三個 | 泛用但不是万能 |
| 本（ほん） | 细长物、瓶、伞、电影/电话等 | ペンを一本、映画を二本 | 读音有 `ぽん/ぼん/ほん` 变化 |
| 枚（まい） | 薄平物 | 紙を三枚、シャツを二枚 | 纸、票、盘子、衣服等 |
| 冊（さつ） | 书、本子 | 本を五冊 | 书籍类 |
| 台（だい） | 机器、车辆 | 車を一台、パソコンを二台 | 机械设备 |
| 匹（ひき） | 小动物 | 猫が二匹 | 读音有 `ぴき/びき/ひき` 变化 |
| 頭（とう） | 大型动物 | 牛が三頭 | 大动物 |
| 羽（わ） | 鸟、兔子 | 鳥が一羽 | 读音有变化 |
| 杯（はい） | 杯、碗、饮品 | コーヒーを一杯 | `いっぱい` 也可表示“很多/满” |
| 回（かい） | 次数 | 三回行きました | 次数 |
| 階（かい） | 楼层 | 三階 | 三階可读 さんがい |
| 番（ばん） | 顺序、号码 | 一番、三番 | 一番也可表示“最” |
| 円（えん） | 日元 | 五百円 | 金额 |
| 年/月/日 | 日期时间 | 2026年、六月、三日 | 日期读法特殊多 |

**常见数字读法**：

| 数字 | 音读 | 和语/特殊 | 例子 |
| ---- | ---- | --------- | ---- |
| 1 | いち | ひとつ、ひとり | 一つ、一人 |
| 2 | に | ふたつ、ふたり | 二つ、二人 |
| 3 | さん | みっつ | 三冊、三つ |
| 4 | よん / し | よっつ | 四人（よにん）、四月（しがつ） |
| 5 | ご | いつつ | 五枚、五つ |
| 6 | ろく | むっつ | 六本（ろっぽん） |
| 7 | なな / しち | ななつ | 七人（ななにん）、七月（しちがつ） |
| 8 | はち | やっつ | 八本（はっぽん） |
| 9 | きゅう / く | ここのつ | 九時（くじ）、九人（きゅうにん） |
| 10 | じゅう | とお | 十冊、十日（とおか） |

**音变高频表**：

| 助数词 | 1 | 3 | 6 | 8 | 10 | 说明 |
| ------ | - | - | - | - | -- | ---- |
| 本 | 一本（いっぽん） | 三本（さんぼん） | 六本（ろっぽん） | 八本（はっぽん） | 十本（じゅっぽん） | は行常变 `ぽん/ぼん` |
| 匹 | 一匹（いっぴき） | 三匹（さんびき） | 六匹（ろっぴき） | 八匹（はっぴき） | 十匹（じゅっぴき） | 小动物 |
| 杯 | 一杯（いっぱい） | 三杯（さんばい） | 六杯（ろっぱい） | 八杯（はっぱい） | 十杯（じゅっぱい） | 注意 `いっぱい` 也有“满/很多” |
| 階 | 一階（いっかい） | 三階（さんがい） | 六階（ろっかい） | 八階（はっかい） | 十階（じゅっかい） | 三階常浊化 |
| 回 | 一回（いっかい） | 三回（さんかい） | 六回（ろっかい） | 八回（はっかい） | 十回（じゅっかい） | 和 階 发音相近，语境区分 |

**数量表达例句**：

| 句子 | 含义 | 说明 |
| ---- | ---- | ---- |
| りんごを三個買いました。 | 买了三个苹果。 | 数量词放动词前，自然 |
| 三個のりんごを買いました。 | 买了三个苹果。 | 数量修饰名词，稍强调数量短语 |
| 学生が五人います。 | 有五个学生。 | 存在句数量词常放谓语前 |
| 一時間ぐらい勉強しました。 | 学了大约一小时。 | 时间长度作状语 |
| 一度だけ行ったことがあります。 | 只去过一次。 | 限定次数 |
| 一人しか来ませんでした。 | 只来了一个人。 | `しか` 配否定 |

### 3.5 形式名词

形式名词保留名词语法功能，但语义较抽象，常把前面的动词句、形容词句、名词句包装成“事情、东西、时候、地方、原因、样子”等。很多高级句型都建立在形式名词上。

| 形式名词 | 核心含义 | 接续 | 例子 | 用法提示 |
| -------- | -------- | ---- | ---- | -------- |
| こと | 事情、行为、抽象内容 | 普通形 + こと | 日本語を勉強することが好きです。 | 抽象、规则、经验、能力常用 |
| の | 事情、行为、具体感知内容 | 普通形 + の | 走っているのを見ました。 | 口语、具体、感官对象常用 |
| もの | 东西、理应、感慨、理由 | 普通形 + もの | 子供はよく泣くものです。 | 常有一般规律或情感色彩 |
| ところ | 地方、阶段、时点 | 普通形 + ところ | 今帰るところです。 | 表动作阶段：将要/正在/刚刚 |
| とき | 时候 | 普通形 + とき | 日本に行ったとき、写真を撮りました。 | 注意前后时态关系按实际判断 |
| ため | 目的、原因、利益 | 普通形/名词の + ため | 合格するために勉強します。 | 目的和原因都可用 |
| はず | 理应、按理说 | 普通形 + はず | 彼は来るはずです。 | 根据事实推断 |
| わけ | 道理、理由、怪不得 | 普通形 + わけ | 今日は休みなわけです。 | 说明逻辑结果 |
| つもり | 打算、自认为 | 辞书形/ない形 + つもり | 明日行くつもりです。 | 说计划或主观认定 |
| まま | 保持原样 | た形/ない形/名词の + まま | 電気をつけたまま寝ました。 | 状态不变 |
| うち | 期间、趁着 | 普通形/名词の + うち | 若いうちに勉強したほうがいい。 | 某状态持续期间 |
| せい | 负面原因 | 普通形/名词の + せい | 雨のせいで遅れました。 | 多为责怪、不好结果 |
| おかげ | 正面原因 | 普通形/名词の + おかげ | 先生のおかげで合格しました。 | 多为感谢、好结果 |
| かわり | 代替、交换 | 普通形/名词の + かわり | 私のかわりに行ってください。 | 代替或补偿关系 |

**`こと` 与 `の` 的常见区别**：

| 用法 | 多用 `こと` | 多用 `の` | 例子 |
| ---- | ---------- | -------- | ---- |
| 抽象行为 | 是 | 可 | 日本語を勉強することは大切です。 |
| 感官动词对象 | 不自然 | 是 | 彼が走っているのを見ました。 |
| 传达/说话内容 | 是 | 较少 | 来週試験があることを伝えました。 |
| 喜好/擅长 | 都可 | 都可，口语自然 | 歌うこと/のが好きです。 |
| 固定句型 | 按句型 | 按句型 | ことがある、ことにする、のだ |

**形式名词常见句型**：

| 句型 | 含义 | 例子 | 注意 |
| ---- | ---- | ---- | ---- |
| `Vたことがある` | 有过某经验 | 日本へ行ったことがあります。 | 经验句型 |
| `Vることができる` | 能做某事 | 漢字を読むことができます。 | 可换成可能形，但语感较正式 |
| `Vることにする` | 决定做 | 毎日運動することにしました。 | 主观决定 |
| `Vることになる` | 结果变成/安排为 | 来月出張することになりました。 | 外部决定或自然结果 |
| `普通形 + のだ/んだ` | 解释、强调背景 | 電車が遅れたんです。 | 口语多用 `んだ/んです` |
| `Vているところ` | 正在做 | 今食べているところです。 | 动作阶段 |
| `Vたばかり` | 刚刚做完 | 今帰ったばかりです。 | 可与实际时间间隔有主观感 |
| `Vたまま` | 做完后保持状态 | 窓を開けたまま寝た。 | 状态未恢复 |

**实用判断**：
  - 看到 `こと、の、もの、ところ` 时，先把前面整段当成一个“名词块”。
  - 形式名词后面常接 `は、が、を、に、で、です`，说明它已经把前句名词化。
  - 不要逐字翻译形式名词；很多时候中文不需要翻出“事情、东西、地方”。
  - 高级句型常是“普通形 + 形式名词 + 助词/助动词”，查表时按整体查。

## 4. 助词系统

助词是日语句子的“关系标记”。汉语和英语常靠词序判断主语、宾语、地点、方向；日语则主要靠名词后的助词判断。学助词时不要只背中文翻译，要同时记住：**前面接什么、标记什么关系、和哪个谓语搭配、是否有语气限制**。

### 4.0 助词总览

| 类型 | 日语名称 | 主要功能 | 常见助词/形式 | 例子 |
| ---- | -------- | -------- | ------------- | ---- |
| 格助词 | 格助詞 | 标记名词和谓语的关系 | `が、を、に、で、へ、と、から、まで、より、の` | 本を読む、学校で勉強する |
| 提示/系助词 | 係助詞・提示助詞 | 提示话题、对比、强调 | `は、も、こそ、しか、さえ` | 私は行く、これしかない |
| 副助词 | 副助詞 | 限定、程度、例示、添加 | `だけ、ばかり、くらい、ほど、など、なんて` | 三つだけ、子供でもできる |
| 并列助词 | 並立助詞 | 并列、列举 | `と、や、たり、だの` | 犬と猫、映画を見たりする |
| 接续助词 | 接続助詞 | 连接分句，表示原因、条件、逆接等 | `て、から、ので、が、けど、のに、ても、ば、たら、なら、と` | 雨なので行かない |
| 终助词 | 終助詞 | 放句末，表示疑问、提醒、共鸣、感叹 | `か、ね、よ、な、ぞ、ぜ、わ` | 行きますか、寒いね |
| 复合助词 | 複合助詞 | 多个词素固定化，表达精细关系 | `について、に対して、を通じて、によって` | 日本語について話す |

**助词叠加顺序常见模式**：

| 模式 | 例子 | 说明 |
| ---- | ---- | ---- |
| 格助词 + は | 東京**では**便利です。 | 把地点、时间、对象等提示为话题或对比 |
| 格助词 + も | 東京**にも**あります。 | 表示“也在/也给/也到”等 |
| 数量/名词 + だけ/しか | 一つ**だけ**ある。一つ**しか**ない。 | 限定范围，`しか` 必须配否定 |
| 名词 + で + も | 子供**でも**できる。 | 举极端例子：连孩子也能 |
| 疑问词 + も + 否定 | 何**も**ない。誰**も**来ない。 | 全面否定 |
| 疑问词 + でも | 何**でも**いい。 | 任意选择 |

**学习助词的实用顺序**：
  - 先掌握 `は、が、を、に、で、へ、と、の、から、まで`。
  - 再掌握限定和强调：`も、だけ、しか、ばかり、こそ、さえ、まで`。
  - 然后按功能整理复合助词：关于、对象、依据、范围、原因、让步、比较、变化。
  - 最后处理句末语气：`ね、よ、か、な` 等。

### 4.1 格助词：标记句子成分

格助词主要接在名词后，说明这个名词和谓语的关系。一个助词往往有多种功能，必须结合谓语判断。

| 助词 | 核心功能 | 典型结构 | 例子 | 注意 |
| ---- | -------- | -------- | ---- | ---- |
| `が` | 主语、新信息、状态对象 | `NがV/Adj` | 雨が降る。日本語が好きです。 | 疑问词主语多用 `が` |
| `を` | 动作对象、经过点、离开点 | `NをV` | 本を読む。橋を渡る。家を出る。 | 不总是“宾语” |
| `に` | 到达点、存在地点、时间点、对象、结果 | `NにV` | 学校に行く。机にある。先生に聞く。 | 功能最多，按谓语判断 |
| `で` | 动作地点、手段、范围、材料、原因 | `NでV` | 図書館で読む。電車で行く。 | 存在地点一般用 `に` |
| `へ` | 方向、朝向 | `NへV` | 日本へ行く。未来へ進む。 | 读作 `え`；强调方向 |
| `と` | 同伴、引用、完全并列、相互动作 | `NとV` | 友達と行く。「はい」と言う。 | 列举完整项时用 `と` |
| `の` | 所属、属性、同位、名词化、从句主语 | `NのN` | 私の本、日本語の先生 | 不要机械翻成“的” |
| `から` | 起点、来源、原因、材料 | `Nから` | 東京から来た。疲れから倒れた。 | 可接句子表原因 |
| `まで` | 终点、范围、极端例子 | `Nまで` | 東京まで行く。子供まで知っている。 | 表极端时接近“连……都” |
| `より` | 比较基准、起点（书面） | `Nより` | 彼より高い。会議は三時より。 | 比较中常和 `ほう` 搭配 |

#### 4.1.1 `は` 与 `が`

严格说 `は` 是提示/系助词，不是格助词；但它和 `が` 是理解句子的第一组核心对比。

| 对比 | `は` | `が` | 例子 |
| ---- | ---- | ---- | ---- |
| 基本功能 | 话题、对比 | 主语、新信息、焦点 | 私は学生です。雨が降る。 |
| 疑问回答 | 回答“关于 X 如何” | 回答“谁/什么” | 田中さんは行きますか。/ 誰が行きますか。 |
| 信息状态 | 已知、拿来谈 | 新出现、被发现 | 昔々、おじいさんがいました。 |
| 强调效果 | 对比其他项目 | 排他：正是它 | 魚は食べるが肉は食べない。私がやります。 |
| 从句中 | 较少统领从句 | 常标从句主语 | 彼が来た時、私は家にいた。 |

| 句子 | 含义 | 说明 |
| ---- | ---- | ---- |
| 私は日本語が好きです。 | 我喜欢日语。 | `私` 是话题，`日本語` 是感受对象 |
| 日本語は好きです。 | 日语我喜欢。 | 可能暗含“其他不一定” |
| 日本語が好きです。 | 喜欢的是日语。/ 我喜欢日语。 | 根据语境可省略话题 |
| 誰が来ましたか。 | 谁来了？ | 疑问词作主语用 `が` |
| 田中さんは来ましたか。 | 田中先生来了吗？ | 田中是话题 |

#### 4.1.2 `に` 与 `で`

`に` 常标“点”：存在点、到达点、时间点、对象点、结果点。`で` 常标“场”：动作发生场所、手段、范围、原因。

| 功能 | `に` | `で` |
| ---- | ---- | ---- |
| 存在地点 | 机の上**に**本があります。 | 不用 `で` |
| 动作地点 | 可用于附着/进入等结果点 | 図書館**で**勉強します。 |
| 到达点 | 学校**に**行きます。 | 不用 `で` |
| 时间点 | 7時**に**起きます。 | 不用 `で` |
| 手段工具 | 不用 `に` | 電車**で**行きます。 |
| 原因 | 状态变化可用 `に`：病気になる | 風邪**で**休みました。 |
| 结果 | 医者**に**なる。 | 不用 `で` |
| 范围 | 不常用 | 日本**で**一番高い山。 |

#### 4.1.3 `を` 的三种高频功能

| 功能 | 结构 | 例子 | 说明 |
| ---- | ---- | ---- | ---- |
| 直接对象 | `NをV` | 本を読む。水を飲む。 | 最常见 |
| 经过点 | `场所を移动V` | 橋を渡る。公園を散歩する。 | 不是把桥当宾语，而是经过桥 |
| 离开点 | `场所を出る/離れる/卒業する` | 家を出る。大学を卒業する。 | 表离开的起点 |

#### 4.1.4 `と`、`や`、`の`

| 助词 | 功能 | 例子 | 区别 |
| ---- | ---- | ---- | ---- |
| `と` | 完全列举 | りんごとバナナを買った。 | 列出的就是这些 |
| `や` | 不完全列举 | りんごやバナナを買った。 | 还有其他类似物 |
| `など` | 举例、泛指 | りんごやバナナなど | “等等、之类” |
| `と` | 同伴 | 友達と行く。 | 和某人一起 |
| `と` | 引用 | 「はい」と言った。 | 引用内容 |
| `の` | 所属、属性 | 私の本、日本の文化 | 表关系，不只等于“的” |
| `の` | 同位/说明 | 友達の田中さん | “作为朋友的田中先生” |

### 4.2 复合助词：按语义功能速查

复合助词多由 `格助词 + 动词/名词/助词` 固定化而成，常用于书面、正式说明和 JLPT 中高级语法。学习时按功能记，比按假名顺序记更实用。

#### 4.2.1 主题、对象、立场

| 表达 | 接续 | 功能 | 例子 | 语体/注意 |
| ---- | ---- | ---- | ---- | --------- |
| `について` | N + について | 关于、就某主题 | 日本文化について研究する。 | 中性常用 |
| `に関して` | N + に関して | 关于、涉及 | この件に関して説明します。 | 较正式 |
| `をめぐって` | N + をめぐって | 围绕争论/讨论 | その問題をめぐって議論が起きた。 | 常用于争议 |
| `に対して` | N + に対して | 对、针对 | 子供に対して優しい。 | 态度、行为对象 |
| `に対する` | N + に対する + N | 对……的 | 提案に対する意見。 | 修饰名词 |
| `にとって` | N + にとって | 对……来说 | 私にとって大切です。 | 评价立场 |
| `として` | N + として | 作为、以身份 | 医者として働く。 | 身份、资格 |
| `としては` | N + としては | 作为……而言 | 初心者としては上手です。 | 限定评价立场 |
| `としても` | N/普通形 + としても | 即使作为/即使 | 彼としても困るでしょう。 | 让步假设 |
| `にしては` | N/普通形 + にしては | 就……而言却 | 子供にしてはしっかりしている。 | 与预期不符 |

| 表达 | 接续 | 功能 | 例子 | 语体/注意 |
| ---- | ---- | ---- | ---- | --------- |
| `によって` | N + によって | 根据、由于、通过、由 | 法律によって決まる。 | 多义，按谓语判断 |
| `に基づいて` | N + に基づいて | 基于、根据 | データに基づいて判断する。 | 书面正式 |
| `をもとに` | N + をもとに | 以……为基础 | 経験をもとに書く。 | 材料、依据 |
| `を踏まえて` | N + を踏まえて | 在考虑……基础上 | 結果を踏まえて修正する。 | 正式 |
| `を通じて` | N + を通じて | 通过、贯穿期间 | 友人を通じて知り合う。 | 媒介/期间 |
| `を通して` | N + を通して | 通过、贯穿 | 体験を通して学ぶ。 | 较口语也可 |
| `をもって` | N + をもって | 以、凭借、到此为止 | 本日をもって終了します。 | 正式公告多 |
| `による` | N + による + N | 由……造成/根据……的 | 地震による被害。 | 修饰名词 |
| `を介して` | N + を介して | 通过、中介 | 代理店を介して購入する。 | 正式 |

| ---- | ---- | ---- | ---- | --------- |

### 4.3 副助词、提示助词：限定、添加、强调

这些助词常接在名词、数量词、部分助词或句子后，改变信息焦点。

| 助词 | 功能 | 例子 | 注意 |
| ---- | ---- | ---- | ---- |
| `も` | 也；连；全面否定 | 私も行く。何もない。 | 疑问词 + も + 否定 = 全否定 |
| `だけ` | 只、仅 | 水だけ飲む。 | 中性限定 |
| `しか` | 只有 | 水しか飲まない。 | 必须配否定 |
| `ばかり` | 净是、刚刚、左右 | 遊んでばかりいる。帰ったばかり。 | 常带偏向或单一感 |
| `のみ` | 仅、只 | 関係者のみ入場可。 | 书面正式 |
| `こそ` | 正是、才 | 今度こそ成功したい。 | 强强调，常带决心或对比 |
| `さえ` | 甚至、连 | 子供さえできる。 | 举极端例子 |
| `すら` | 甚至、连 | 名前すら忘れた。 | 比 `さえ` 更书面/强烈 |
| `まで` | 甚至到、连 | 親まで反対した。 | 表极端范围 |
| `でも` | 即使、连、随便 | 子供でも分かる。何でもいい。 | 举例或任意 |
| `など` | 等等、之类 | 本や雑誌など。 | 举例，不完全列举 |
| `なんて` | ……之类；竟然 | そんなことなんて信じない。 | 口语，轻视/惊讶 |
| `くらい/ぐらい` | 大约、程度、至少 | 10分くらい。これくらいできる。 | 数量或程度 |
| `ほど` | 程度、越……越、比较基准 | 泣きたいほど痛い。 | 程度较强 |
| `きり` | 只有；之后一直 | 一度会ったきり。 | 动作后状态不变 |
| `だらけ` | 满是 | 間違いだらけ。 | 负面多 |
| `ずつ` | 每、各 | 一人ずつ入る。 | 分配 |
| `ごと` | 每 | 一日ごとに変わる。 | 周期、单位 |

**限定助词辨析**：

| 对比 | 例子 | 语感 |
| ---- | ---- | ---- |
| `だけ` | りんごだけ食べます。 | 只吃苹果，中性 |
| `しか` | りんごしか食べません。 | 除苹果外不吃，带“少/不够”感 |
| `ばかり` | りんごばかり食べています。 | 老是吃苹果，可能带负面评价 |
| `のみ` | 会員のみ利用できます。 | 仅限会员，正式公告 |

### 4.4 并列助词、引用助词与语气助词

#### 4.4.1 并列和列举

| 形式 | 功能 | 例子 | 注意 |
| ---- | ---- | ---- | ---- |
| `AとB` | 完全列举 | 父と母が来ます。 | 列出的就是这些 |
| `AやB` | 不完全列举 | 本や雑誌を読む。 | “A、B之类” |
| `AとかBとか` | 口语列举 | 映画とか音楽とかが好き。 | 口语、举例 |
| `AだのBだの` | 列举并带不满/随意 | 忙しいだの疲れただの言う。 | 常带抱怨 |
| `VたりVたりする` | 动作列举、时而 | 映画を見たり本を読んだりする。 | 动词用た形 + り |
| `AもBも` | A和B都 | 父も母も来ます。 | 全部肯定或否定 |
| `AかB` | A或B | 今日か明日行きます。 | 选择 |

#### 4.4.2 引用和内容

| 形式 | 功能 | 例子 | 常搭配谓语 |
| ---- | ---- | ---- | ---------- |
| `と` | 直接/间接引用 | 「行きます」と言った。 | 言う、思う、考える、聞く |
| `って` | 口语引用/话题 | 明日来るって聞いた。 | 言う、聞く、思う；口语 |
| `という` | 叫作、名为、内容说明 | 田中という人。合格したという知らせ。 | 修饰名词 |
| `との` | 书面引用修饰 | 参加するとの連絡。 | 新闻、报告 |
| `か` | 嵌入疑问 | いつ来るか分からない。 | 分かる、知る、聞く |
| `かどうか` | 是否 | 行くかどうか決める。 | 二选一不确定 |

#### 4.4.3 句末语气

| 终助词 | 功能 | 例子 | 语气 |
| ------ | ---- | ---- | ---- |
| `か` | 疑问 | 行きますか。 | 礼貌疑问；常体中可省略上升调 |
| `ね` | 共鸣、确认 | 寒いですね。 | “是吧/呢” |
| `よ` | 提醒、告知、强调 | もう時間ですよ。 | 告诉对方不知道的信息 |
| `よね` | 确认并期待同意 | 明日ですよね。 | 比单独 `ね` 更明确 |
| `な` | 感叹、自言自语；禁止 | きれいだな。行くな。 | 禁止时接辞书形 |
| `かな` | 自问、委婉疑问 | 大丈夫かな。 | 柔和、不确定 |
| `ぞ` | 强提醒、男性/粗硬 | 行くぞ。 | 学习者慎用 |
| `ぜ` | 口语强调、男性/随意 | すごいぜ。 | 非正式 |
| `わ` | 柔和感叹/强调 | そうだわ。 | 地域和性别语感复杂 |
| `っけ` | 回忆确认 | 今日は何曜日だっけ。 | “来着？” |

### 4.5 接续助词：连接分句

接续助词把两个句子或成分连接起来，表达原因、条件、逆接、让步、并列、时间等关系。它们常接在动词、形容词、名词谓语的活用形后。

| 功能 | 形式 | 接续要点 | 例子 | 注意 |
| ---- | ---- | -------- | ---- | ---- |
| 顺接/并列 | `て` | て形 | ご飯を食べて寝た。 | 可表顺序、并列、轻原因 |
| 完成后 | `てから` | て形 + から | 宿題をしてから寝る。 | 强调前项完成后 |
| 原因 | `から` | 普通形/敬体均可 | 忙しいから行けない。 | 主观、直接 |
| 原因 | `ので` | 普通形；名词/な形容词 + なので | 雨なので行かない。 | 较客观、柔和 |
| 原因解释 | `ものだから` | 普通形 | 寝坊したものだから遅れた。 | 解释、辩解语气 |
| 逆接 | `が` | 普通形/敬体 | 行きたいが時間がない。 | 书面/正式也常用 |
| 逆接 | `けど/けれども` | 普通形/敬体 | 高いけど買った。 | 口语到正式都有 |
| 逆接 | `のに` | 普通形；名词/な形容词 + なのに | 勉強したのに落ちた。 | 带意外、不满 |
| 让步 | `ても/でも` | て形 | 雨でも行く。 | 即使 |
| 并行动作 | `ながら` | ます形词干 | 音楽を聞きながら勉強する。 | 同一主体为原则 |
| 让步 | `ながらも` | ます形词干/形容词等 | 若いながらも立派だ。 | 虽然却 |
| 反复列举 | `たり` | た形 + り | 見たり聞いたりする。 | 后面常接 `する` |
| 条件 | `と` | 辞书形/ない形 | 春になると花が咲く。 | 自然结果、习惯；不能接意志命令 |
| 条件 | `ば` | 条件形 | 安ければ買う。 | 条件关系较一般 |
| 条件 | `たら` | た形 + ら | 雨だったら行かない。 | 一次性、口语常用 |
| 条件/话题 | `なら` | 普通形/名词 | 日本へ行くなら春がいい。 | 承接对方话题 |
| 目的 | `ように` | 辞书形/ない形 | 忘れないようにメモする。 | 常用于非意志动词/可能 |
| 程度结果 | `ほど` | 普通形/名词 | 泣きたいほど痛い。 | 程度 |
| 趁机 | `ついでに` | 名词の/Vる・Vた | 買い物のついでに寄る。 | 顺便 |

**原因 `から/ので/ため` 简辨**：

| 形式 | 语感 | 例子 | 注意 |
| ---- | ---- | ---- | ---- |
| `から` | 主观、直接、口语自然 | 暑いから窓を開ける。 | 可接命令、请求、意志 |
| `ので` | 客观、柔和、礼貌 | 暑いので窓を開けます。 | 请求中更委婉 |
| `ため` | 书面、正式、结果说明 | 台風のため中止です。 | 公告、新闻常用 |

**条件 `と/ば/たら/なら` 简辨**：

| 形式 | 核心用法 | 例子 | 限制/语感 |
| ---- | -------- | ---- | --------- |
| `と` | 一……就；自然规律、操作结果 | ボタンを押すと開く。 | 后项一般不用意志、命令、请求 |
| `ば` | 如果条件成立 | 安ければ買います。 | 前项状态条件明显 |
| `たら` | 如果/做完后 | 雨が降ったら中止です。 | 最通用，口语自然 |
| `なら` | 如果说到…… | 日本に行くなら京都がいい。 | 承接话题、建议 |

### 4.6 助词辨析与常见错误

| 易混点 | 错误/不自然 | 自然表达 | 说明 |
| ------ | ----------- | -------- | ---- |
| 存在地点 | 図書館で本があります。 | 図書館に本があります。 | 存在地点用 `に` |
| 动作地点 | 図書館に勉強します。 | 図書館で勉強します。 | 动作发生地点用 `で` |
| 到达点 | 学校で行きます。 | 学校に/へ行きます。 | 移动到达点用 `に/へ` |
| 时间点 | 昨日に行きました。 | 昨日行きました。 | 相对时间词常不用 `に`：昨日、今日、明日 |
| 具体时间 | 7時起きます。 | 7時に起きます。 | 明确时间点常用 `に` |
| 宾语 | 本が読みます。 | 本を読みます。 | 读书的对象用 `を` |
| 好恶对象 | 日本語を好きです。 | 日本語が好きです。 | 好き、嫌い、上手、下手 的对象常用 `が` |
| 能力对象 | 日本語をできます。 | 日本語ができます。 | 能力对象常用 `が` |
| 完全/不完全列举 | りんごやバナナを買った（只有两样）。 | りんごとバナナを買った。 | 只有列出的项目用 `と` |
| 限定 | りんごしか食べます。 | りんごしか食べません。 | `しか` 必须配否定 |
| 话题过度使用 | 私は私は毎日勉強します。 | 私は毎日勉強します。 | `は` 不要重复堆叠 |
| 助词遗漏 | 私 学校 行きます。 | 私は学校に行きます。 | 入门阶段先写全助词 |

**选择助词的四步法**：
  - 先看谓语：存在、移动、动作、感受、能力、判断分别需要不同助词。
  - 再问名词角色：主体、对象、地点、方向、手段、时间、范围、原因。
  - 再看信息结构：要话题/对比用 `は`，要新信息/焦点用 `が`。
  - 最后看语体和固定搭配：正式复合助词、口语省略、惯用表达要按整体记。

## 5. 谓语活用与助动词

谓语是日语句子的发动机。一个句子的时态、否定、礼貌、可能、被动、使役、意志、命令等，大多集中在句末谓语上。学习顺序应是：**判断谓语类型 → 判断活用类别 → 选正确活用形 → 接助动词或句型**。

### 5.1 活用总览

| 谓语类型 | 是否活用 | 基本形 | 礼貌形 | 否定 | 过去 | 接续形 |
| -------- | -------- | ------ | ------ | ---- | ---- | ------ |
| 五段动词 | 活用 | 書く | 書きます | 書かない | 書いた | 書いて |
| 一段动词 | 活用 | 食べる | 食べます | 食べない | 食べた | 食べて |
| サ变动词 | 不规则 | する | します | しない | した | して |
| カ变动词 | 不规则 | 来る | 来ます | 来ない | 来た | 来て |
| い形容词 | 活用 | 高い | 高いです | 高くない | 高かった | 高くて |
| な形容词 | 准活用 | 静かだ | 静かです | 静かではない | 静かだった | 静かで |
| 名词谓语 | 准活用 | 学生だ | 学生です | 学生ではない | 学生だった | 学生で |

**先看句末，再拆后缀**：

| 句末 | 拆解 | 语法意义 |
| ---- | ---- | -------- |
| 読みませんでした | 読み + ません + でした | 礼貌否定过去 |
| 読まなければならない | 読ま + なければ + ならない | 必须读 |
| 読ませられました | 読ま + せ + られ + ました | 被迫读/使役被动礼貌过去 |
| 高くなかったです | 高く + なかった + です | い形容词否定过去礼貌 |
| 静かではありませんでした | 静か + では + ありません + でした | な形容词/名词否定过去礼貌 |

**活用形的实际用途**：

| 活用形 | 主要用途 | 例子 |
| ------ | -------- | ---- |
| 辞书形 | 查词典、普通非过去、接名词/こと/つもり | 行く、食べる、読む本 |
| ます形 | 礼貌叙述；去 `ます` 后接句型 | 行きます、食べに行く、読みたい |
| ない形 | 否定；接义务、禁止、目的 | 行かない、食べなければならない |
| て形 | 接续、请求、进行、状态、许可 | 行ってください、食べている |
| た形 | 过去、经验、条件、后续 | 行ったことがある、食べたら |
| 条件形 | 如果……就 | 行けば、食べれば |
| 意志形 | 意志、提议 | 行こう、食べよう |
| 命令形 | 命令、口号、引用命令 | 行け、食べろ |
| 可能形 | 能做 | 行ける、食べられる |
| 被动形 | 被、受影响、尊敬 | 言われる、読まれる |
| 使役形 | 让、使 | 行かせる、食べさせる |

### 5.2 动词分类：五段、一段、サ变、カ变

动词分类决定活用。不能只看中文意思，要看日语辞书形词尾。

| 类别 | 辞书形特征 | ます形 | ない形 | 例子 | 判断要点 |
| ---- | ---------- | ------ | ------ | ---- | -------- |
| 五段动词 | 词尾在う段，活用时跨五段 | 書きます | 書かない | 書く、話す、読む、買う | 词尾变化成 あ/い/う/え/お 段 |
| 一段动词 | 多以 `いる/える` 结尾，去 `る` 接后续 | 食べます | 食べない | 食べる、見る、起きる | 去 `る` 后直接接 `ます/ない/た` |
| サ变动词 | `する` 及 `Nする` | します | しない | する、勉強する | `する` 变 `し` |
| カ变动词 | 只有 `来る` | 来ます | 来ない | 来る | 读音变化：くる、きます、こない |

**一段动词和五段动词的陷阱**：很多 `〜いる/〜える` 是一段，但不是全部。

| 动词 | 类别 | ます形 | ない形 | 备注 |
| ---- | ---- | ------ | ------ | ---- |
| 食べる | 一段 | 食べます | 食べない | 典型一段 |
| 見る | 一段 | 見ます | 見ない | 只有一个假名词干 |
| 起きる | 一段 | 起きます | 起きない | いる结尾一段 |
| 帰る | 五段 | 帰ります | 帰らない | 例外，不能按一段处理 |
| 入る | 五段 | 入ります | 入らない | 例外 |
| 走る | 五段 | 走ります | 走らない | 例外 |
| 知る | 五段 | 知ります | 知らない | 例外 |
| 要る | 五段 | 要ります | 要らない | 例外 |
| 切る | 五段 | 切ります | 切らない | 例外 |
| 減る | 五段 | 減ります | 減らない | 例外 |

**快速判断流程**：
  - 如果是 `する/来る`，先按不规则处理。
  - 如果不是 `る` 结尾，通常是五段：書く、話す、読む、買う。
  - 如果是 `〜ある/〜うる/〜おる` 结尾，通常是五段。
  - 如果是 `〜いる/〜える`，大多是一段，但常用例外要单独记。
  - 查词典时最好同时记 `辞书形 + ます形 + ない形`。

| `う、つ、る` | `って` | `った` | 買って、待って、帰って |

#### 5.3.2 一段、不规则动词活用总表

| 类别 | 辞书形 | ます形 | ない形 | て形 | た形 | 条件形 | 意志形 | 命令形 | 可能形 |
| ---- | ------ | ------ | ------ | ---- | ---- | ------ | ------ | ------ | ------ |
| 一段 | 食べる | 食べます | 食べない | 食べて | 食べた | 食べれば | 食べよう | 食べろ/食べよ | 食べられる/食べれる |
| 一段 | 見る | 見ます | 見ない | 見て | 見た | 見れば | 見よう | 見ろ/見よ | 見られる/見れる |
| サ变 | する | します | しない | して | した | すれば | しよう | しろ/せよ | できる |
| サ变复合 | 勉強する | 勉強します | 勉強しない | 勉強して | 勉強した | 勉強すれば | 勉強しよう | 勉強しろ | 勉強できる |
| カ变 | 来る | 来ます | 来ない | 来て | 来た | 来れば | 来よう | 来い | 来られる/来れる |

**ら抜き言葉**：一段动词和 `来る` 的可能形，规范形式是 `食べられる、見られる、来られる`。口语中常听到 `食べれる、見れる、来れる`，称为 `ら抜き言葉`。正式写作、考试和商务中优先用规范形式。

### 5.4 ます形、辞书形与连用形

| 形式 | 生成方式 | 功能 | 例子 |
| ---- | -------- | ---- | ---- |
| 辞书形 | 词典基本形 | 普通非过去、修饰名词、接句型 | 明日行く。読む本。行くつもり。 |
| ます形 | 连用形 + ます | 礼貌叙述 | 毎日勉強します。 |
| ます形词干 | 去掉 `ます` | 接 `たい、ながら、に行く、やすい` 等 | 読みたい、食べながら、買いに行く |
| 连用形 | 五段い段/一段词干/し/来（き） | 中止、复合、名词化 | 書き、食べ、し、来 |

**ます形词干常用句型**：

| 句型 | 含义 | 例子 | 注意 |
| ---- | ---- | ---- | ---- |
| `Vます形词干 + たい` | 想做 | 日本へ行きたいです。 | `たい` 按い形容词活用 |
| `Vます形词干 + ながら` | 一边……一边 | 音楽を聞きながら勉強する。 | 通常同一主体 |
| `Vます形词干 + に行く/来る` | 去/来做某事 | 映画を見に行く。 | 表目的 |
| `Vます形词干 + やすい` | 容易做 | この本は読みやすい。 | 形成い形容词 |
| `Vます形词干 + にくい` | 难做 | この字は読みにくい。 | 形成い形容词 |
| `Vます形词干 + 方` | 做法 | 読み方、使い方 | 名词化 |
| `お + Vます形词干 + ください` | 尊敬请求 | お待ちください。 | 和语动词常用 |
| `ご + サ变名词 + ください` | 尊敬请求 | ご確認ください。 | 汉语サ变名词常用 |

**辞书形常用句型**：

| 句型 | 含义 | 例子 |
| ---- | ---- | ---- |
| `Vる + こと` | 做某事 | 日本語を話すことは難しい。 |
| `Vる + つもり` | 打算 | 来年留学するつもりです。 |
| `Vる + 前に` | 做……之前 | 寝る前に歯を磨く。 |
| `Vる + ために` | 为了 | 合格するために勉強する。 |
| `Vる + のが好き` | 喜欢做 | 歌うのが好きです。 |
| `Vる + べき` | 应该 | 約束を守るべきです。 |

### 5.5 て形、た形与接续形

て形是日语最重要的连接形。它不仅表示“然后”，还用于请求、许可、禁止、进行、状态、完成、授受等。

| て形句型 | 功能 | 例子 | 说明 |
| -------- | ---- | ---- | ---- |
| `Vて、V` | 顺序/并列 | 朝起きて、顔を洗う。 | 前后动作连接 |
| `Vてください` | 请求 | ここに書いてください。 | 礼貌请求 |
| `Vてもいい` | 许可 | 入ってもいいですか。 | 可以…… |
| `Vてはいけない` | 禁止 | ここで写真を撮ってはいけない。 | 不能…… |
| `Vている` | 进行/状态 | 今読んでいる。結婚している。 | 动作进行或结果状态 |
| `Vてある` | 人为结果状态 | 窓が開けてある。 | 某人预先做了，结果保留 |
| `Vてしまう` | 完成/遗憾 | 宿題を忘れてしまった。 | 做完；不小心 |
| `Vておく` | 预先做 | 予約しておく。 | 为以后准备 |
| `Vてみる` | 试着做 | 食べてみる。 | 尝试 |
| `Vてくる` | 做完来；变化到现在 | 買ってくる。寒くなってきた。 | 空间或时间接近 |
| `Vていく` | 做着去；变化向未来 | 歩いていく。増えていく。 | 空间或时间远离 |
| `Vてあげる/くれる/もらう` | 授受 | 手伝ってあげる。 | 带受益方向 |

**た形常用句型**：

| た形句型 | 功能 | 例子 | 说明 |
| -------- | ---- | ---- | ---- |
| `Vた` | 过去/完成 | 昨日映画を見た。 | 过去或完成 |
| `Vたことがある` | 经验 | 日本へ行ったことがある。 | 曾经做过 |
| `Vた後で` | 之后 | 食べた後で寝る。 | 前项完成后 |
| `Vたら` | 如果/之后 | 雨が降ったら行かない。 | 条件或时间先后 |
| `VたりVたりする` | 列举动作 | 掃除したり洗濯したりする。 | 举例，不穷尽 |
| `Vたばかり` | 刚刚 | 起きたばかりです。 | 主观“刚” |
| `Vたまま` | 保持状态 | 電気をつけたまま寝た。 | 没恢复原状 |
| `Vたほうがいい` | 最好 | 早く寝たほうがいい。 | 建议 |

### 5.6 ない形、否定与过去否定

| 类别 | 辞书形 | ない形 | 过去否定 | 例子 |
| ---- | ------ | ------ | -------- | ---- |
| 五段 | 書く | 書かない | 書かなかった | 手紙を書かない。 |
| 五段う结尾 | 買う | 買わない | 買わなかった | `う` 变 `わ` |
| 一段 | 食べる | 食べない | 食べなかった | 去 `る` + ない |
| サ变 | する | しない | しなかった | 勉強しない |
| カ变 | 来る | 来ない（こない） | 来なかった | 読音注意：こない |
| い形容词 | 高い | 高くない | 高くなかった | `い` → `くない` |
| な形容词 | 静かだ | 静かではない | 静かではなかった | 名词同样 |

**ない形常用句型**：

| 句型 | 含义 | 例子 | 注意 |
| ---- | ---- | ---- | ---- |
| `Vないでください` | 请不要 | ここで吸わないでください。 | 礼貌禁止/请求 |
| `Vなくてもいい` | 不做也可以 | 来なくてもいいです。 | 不必 |
| `Vなければならない` | 必须 | 勉強しなければならない。 | 书面/正式常用 |
| `Vないといけない` | 必须 | 早く寝ないといけない。 | 口语自然 |
| `Vなくてはいけない` | 必须 | 行かなくてはいけない。 | 中性 |
| `Vないようにする` | 尽量不 | 忘れないようにする。 | 努力避免 |
| `Vないで` | 不做前项而做后项 | 朝ご飯を食べないで出た。 | “不……就……” |
| `Vずに` | 不做前项而做后项 | 食べずに出た。 | 书面一点；する→せずに |

**否定形式辨析**：

| 形式 | 例子 | 语感 |
| ---- | ---- | ---- |
| `行かない` | 明日行かない。 | 常体否定 |
| `行きません` | 明日行きません。 | 礼貌否定 |
| `行かなかった` | 昨日行かなかった。 | 常体过去否定 |
| `行きませんでした` | 昨日行きませんでした。 | 礼貌过去否定 |
| `行けない` | 忙しくて行けない。 | 不能去（可能否定） |
| `行かれない` | 较少作普通可能否定 | 可能/被动歧义大，现代口语少用 |

### 5.7 可能、被动、使役与使役被动

#### 5.7.1 可能形

| 类别 | 规则 | 例子 | 说明 |
| ---- | ---- | ---- | ---- |
| 五段 | う段 → え段 + る | 書く→書ける、読む→読める | 变成一段式活用 |
| 一段 | 去 `る` + られる | 食べる→食べられる | 口语有ら抜き：食べれる |
| サ变 | する→できる | 勉強する→勉強できる | `Nができる` 也常用 |
| カ变 | 来る→来られる | 来られる | 口语有 来れる |

| 结构 | 例子 | 注意 |
| ---- | ---- | ---- |
| `Nが V可能` | 日本語が読めます。 | 可能对象常用 `が` |
| `Nを V可能` | 日本語を読めます。 | 也可见，偏动作对象感 |
| `Vることができる` | 日本語を読むことができます。 | 较正式、说明性强 |

#### 5.7.2 被动形

| 类别 | 规则 | 例子 |
| ---- | ---- | ---- |
| 五段 | う段 → あ段 + れる | 書く→書かれる、言う→言われる |
| 一段 | 去 `る` + られる | 食べる→食べられる |
| サ变 | する→される | 注意する→注意される |
| カ变 | 来る→来られる | 来られる |

| 被动类型 | 例子 | 说明 |
| -------- | ---- | ---- |
| 直接被动 | 私は先生に褒められた。 | 我被老师表扬 |
| 间接受害被动 | 雨に降られた。 | 被雨淋了，带受害感 |
| 所有物被动 | 財布を盗まれた。 | 钱包被偷，主体受影响 |
| 尊敬 | 先生が来られました。 | `られる` 可表尊敬，需按语境判断 |

#### 5.7.3 使役形

| 类别 | 规则 | 例子 |
| ---- | ---- | ---- |
| 五段 | う段 → あ段 + せる | 書く→書かせる、行く→行かせる |
| 一段 | 去 `る` + させる | 食べる→食べさせる |
| サ变 | する→させる | 勉強する→勉強させる |
| カ变 | 来る→来させる（こさせる） | 来させる |

| 结构 | 例子 | 说明 |
| ---- | ---- | ---- |
| `人に Vさせる` | 子供に野菜を食べさせる。 | 让某人做某事 |
| `人を 自動詞使役` | 子供を走らせる。 | 使某人进行自动作 |
| `Vさせてください` | 休ませてください。 | 请允许我…… |
| `Vさせていただく` | 説明させていただきます。 | 谦逊、正式发言常用 |

#### 5.7.4 使役被动形

| 类别 | 规则 | 例子 | 说明 |
| ---- | ---- | ---- | ---- |
| 五段 | あ段 + せられる | 書く→書かせられる | 被迫做 |
| 五段缩约 | あ段 + される | 書く→書かされる | 口语常见；す结尾不用此缩约 |
| 一段 | 去 `る` + させられる | 食べる→食べさせられる | 被迫吃 |
| サ变 | させられる | 勉強させられる | 被迫学习 |
| カ变 | 来させられる | 来させられる | 被迫来 |

例：子供のころ、毎日ピアノを練習させられました。  
含义：小时候每天被迫练钢琴。

### 5.8 意志、命令、禁止与请求

| 功能 | 形式 | 例子 | 语气 |
| ---- | ---- | ---- | ---- |
| 意志/提议 | 意志形 | 行こう。食べよう。 | “去吧/我打算去” |
| 礼貌提议 | ます形 + ましょう | 行きましょう。 | “一起……吧” |
| 建议询问 | ます形 + ましょうか | 手伝いましょうか。 | “我来……吧？” |
| 命令 | 命令形 | 行け。食べろ。 | 强硬，慎用 |
| 禁止 | 辞书形 + な | 行くな。 | 强硬禁止 |
| 请求 | て形 + ください | 来てください。 | 礼貌请求 |
| 否定请求 | ないでください | 行かないでください。 | 请不要 |
| 许可请求 | てもいいですか | 入ってもいいですか。 | 可以……吗 |
| 委婉请求 | ていただけませんか | 教えていただけませんか。 | 更礼貌 |
| 轻请求 | てくれる？ | 手伝ってくれる？ | 朋友间 |

**命令和请求的安全度**：

| 安全度 | 表达 | 场景 |
| ------ | ---- | ---- |
| 安全 | 〜てください | 普通礼貌请求 |
| 更礼貌 | 〜ていただけませんか | 对上级、客户、陌生人 |
| 亲近 | 〜てくれる？ | 朋友、家人 |
| 强硬 | 命令形、辞书形+な | 口号、上级对下级、紧急情况 |
| 间接 | 〜てもらえますか | 请求帮助，较自然 |

### 5.9 形容词活用形速查

#### 5.9.1 い形容词

| 活用形 | 形式 | 例子（高い） | 用途 |
| ------ | ---- | ------------ | ---- |
| 原形 | `〜い` | 高い | 非过去肯定、修饰名词 |
| 礼貌 | `〜いです` | 高いです | 礼貌肯定 |
| 否定 | `〜くない` | 高くない | 否定 |
| 礼貌否定 | `〜くないです / 〜くありません` | 高くないです / 高くありません | 礼貌否定 |
| 过去 | `〜かった` | 高かった | 过去肯定 |
| 礼貌过去 | `〜かったです` | 高かったです | 礼貌过去 |
| 过去否定 | `〜くなかった` | 高くなかった | 过去否定 |
| て形 | `〜くて` | 高くて | 接续、原因、并列 |
| 连用形 | `〜く` | 高く売る、高くなる | 修饰谓语、变化 |
| 条件形 | `〜ければ` | 高ければ | 如果…… |
| 副词化 | `〜く` | 早く歩く | 修饰动词 |
| 名词化 | `〜さ` | 高さ、寒さ | 程度、性质名词 |
| 变化 | `〜くなる` | 高くなる | 变得…… |
| 使成 | `〜くする` | 部屋を暖かくする | 使其变得…… |

**例外**：`いい` 的活用按 `よい`：いい → よくない、よかった、よければ。

#### 5.9.2 な形容词

| 活用形 | 形式 | 例子（静か） | 用途 |
| ------ | ---- | ------------ | ---- |
| 原形/常体 | `〜だ` | 静かだ | 非过去肯定谓语 |
| 礼貌 | `〜です` | 静かです | 礼貌肯定 |
| 修饰名词 | `〜な + 名词` | 静かな町 | 连体修饰 |
| 修饰谓语 | `〜に + 动词` | 静かに話す | 连用修饰 |
| 否定 | `〜ではない / じゃない` | 静かではない | 否定 |
| 礼貌否定 | `〜ではありません` | 静かではありません | 礼貌否定 |
| 过去 | `〜だった` | 静かだった | 过去肯定 |
| 礼貌过去 | `〜でした` | 静かでした | 礼貌过去 |
| 过去否定 | `〜ではなかった` | 静かではなかった | 过去否定 |
| て形 | `〜で` | 静かで | 接续、原因、并列 |
| 条件形 | `〜なら / 〜であれば` | 静かなら | 如果…… |
| 变化 | `〜になる` | 静かになる | 变得…… |
| 使成 | `〜にする` | 部屋を静かにする | 使其变得…… |

**い形容词与な形容词常见错误**：

| 错误 | 正确 | 原因 |
| ---- | ---- | ---- |
| 静か町 | 静かな町 | な形容词修饰名词要加 `な` |
| 高いな本 | 高い本 | い形容词直接接名词 |
| 静かく話す | 静かに話す | な形容词修饰动词用 `に` |
| 高いになる | 高くなる | い形容词变化用 `くなる` |
| いいくない | よくない | `いい` 按 `よい` 活用 |

### 5.10 名词谓语与な形容词谓语

名词谓语和な形容词谓语活用方式很接近，都靠 `だ/です/で/ではない/だった` 这一套。区别在于：な形容词可以修饰名词时接 `な`，名词一般用 `の`。

| 功能 | 名词 | な形容词 | 例子 |
| ---- | ---- | -------- | ---- |
| 作谓语常体 | 学生だ | 静かだ | 彼は学生だ。部屋は静かだ。 |
| 作谓语敬体 | 学生です | 静かです | 彼は学生です。部屋は静かです。 |
| 否定 | 学生ではない | 静かではない | 学生ではない。静かではない。 |
| 过去 | 学生だった | 静かだった | 学生だった。静かだった。 |
| 接续 | 学生で | 静かで | 学生で、親切です。静かで、便利です。 |
| 修饰名词 | 学生の生活 | 静かな町 | 名词多用 `の`，な形容词用 `な` |
| 修饰谓语 | 学生として働く | 静かに話す | 名词需助词/复合形式；な形容词用 `に` |

| 形式 | 用途 | 例子 |
| ---- | ---- | ---- |
| `Nで` | 并列、原因、中止 | 彼は学生で、妹は会社員です。 |
| `Nではなく` | 不是……而是…… | これは本ではなく雑誌です。 |
| `Nなら` | 如果是…… | 学生なら安くなります。 |
| `Nとして` | 作为…… | 教師として働く。 |
| `Nになる` | 变成…… | 医者になる。 |
| `Nにする` | 决定选…… | コーヒーにします。 |

### 5.11 助动词速查

助动词接在动词、形容词、名词谓语等后面，表达否定、过去、礼貌、推量、被动、使役、希望等。很多助动词本身也会继续活用。

| 助动词/形式 | 接续 | 功能 | 例子 | 注意 |
| ----------- | ---- | ---- | ---- | ---- |
| `ます` | ます形词干 | 礼貌 | 行きます | 否定：行きません |
| `です` | 名词/形容词等 | 礼貌判断 | 学生です、高いです | 不直接接普通动词辞书形作礼貌 |
| `ない` | ない形词干/形容词く | 否定 | 行かない、高くない | 按い形容词活用 |
| `た` | た形 | 过去/完成 | 行った、食べた | 可接句型：たら、たり |
| `たい` | ます形词干 | 想做 | 行きたい | 按い形容词活用；对象可用 `が/を` |
| `れる/られる` | 动词未然形 | 被动、可能、尊敬、自发 | 読まれる、食べられる | 一段可能和被动同形 |
| `せる/させる` | 动词未然形 | 使役 | 行かせる、食べさせる | 让/使 |
| `そうだ` | ます形词干/形容词词干 | 样态 | 雨が降りそうだ。おいしそう。 | いい→よさそう；ない→なさそう |
| `そうだ` | 普通形 | 传闻 | 雨が降るそうだ。 | 不同于样态接续 |
| `ようだ` | 普通形/名词の/な形な | 比况、推测 | 夢のようだ。雨のようだ。 | 较书面 |
| `らしい` | 普通形/名词 | 传闻、典型 | 彼は来るらしい。学生らしい。 | “像个……”也可 |
| `だろう/でしょう` | 普通形 | 推量 | 明日は雨でしょう。 | `でしょう` 较礼貌 |
| `べきだ` | 辞书形 | 应该 | 約束を守るべきだ。 | する→するべき/すべき |
| `ようにする` | 辞书形/ない形 | 努力做到/避免 | 毎日読むようにする。 | 习惯化努力 |
| `ことになる` | 普通形 + こと | 外部决定/结果 | 出張することになった。 | 非自己单方面决定 |
| `ことにする` | 普通形 + こと | 自己决定 | 毎日走ることにした。 | 主观决定 |

**助动词链示例**：

| 形式 | 拆解 | 含义 |
| ---- | ---- | ---- |
| 読みたくない | 読み + たい + ない | 不想读 |
| 読まなかった | 読ま + ない + た | 没读 |
| 読ませたい | 読ま + せる + たい | 想让读 |
| 読ませられた | 読ま + せる + られる + た | 被迫读了 |
| 読まれるでしょう | 読ま + れる + でしょう | 大概会被读/尊敬语境下“读吧” |

### 5.12 自动词、他动词与移动动词

自动词描述主体自身发生变化或状态；他动词描述主体对对象施加动作。很多日语动词成对出现，搭配的助词和语感不同。

| 类型 | 结构 | 例子 | 说明 |
| ---- | ---- | ---- | ---- |
| 自动词 | `Nが 自動詞` | ドアが開く。 | 门自己开了/处于打开变化 |
| 他动词 | `Nを 他動詞` | ドアを開ける。 | 某人把门打开 |
| 自动词结果状态 | `Nが Vている` | 窓が開いている。 | 窗户开着 |
| 他动词人为结果 | `Nが Vてある` | 窓が開けてある。 | 有人事先开好了窗 |

**常见自动词/他动词对**：

| 自动词 | 他动词 | 自动词例句 | 他动词例句 |
| ------ | ------ | ---------- | ---------- |
| 開く（あく） | 開ける（あける） | ドアが開く。 | ドアを開ける。 |
| 閉まる | 閉める | 店が閉まる。 | 店を閉める。 |
| つく | つける | 電気がつく。 | 電気をつける。 |
| 消える | 消す | 火が消える。 | 火を消す。 |
| 入る | 入れる | 水が入る。 | 水を入れる。 |
| 出る | 出す | お金が出る。 | お金を出す。 |
| 始まる | 始める | 会議が始まる。 | 会議を始める。 |
| 終わる | 終える/終わらせる | 授業が終わる。 | 仕事を終える。 |
| 決まる | 決める | 予定が決まる。 | 予定を決める。 |
| 直る | 直す | パソコンが直る。 | パソコンを直す。 |
| 壊れる | 壊す | 皿が壊れる。 | 皿を壊す。 |
| 落ちる | 落とす | 財布が落ちる。 | 財布を落とす。 |
| 集まる | 集める | 人が集まる。 | 人を集める。 |
| 変わる | 変える | 予定が変わる。 | 予定を変える。 |

**移动动词和助词**：

| 动词类型 | 常用助词 | 例子 | 说明 |
| -------- | -------- | ---- | ---- |
| 到达 | `に/へ` | 学校に行く。東京へ向かう。 | `に` 重到达点，`へ` 重方向 |
| 出发/来源 | `から` | 家から出る。日本から来る。 | 起点 |
| 经过 | `を` | 橋を渡る。道を歩く。 | 经过空间 |
| 移动范围 | `を` | 公園を散歩する。 | 在范围内移动 |
| 目的 | `ます形词干 + に` | 本を買いに行く。 | 去做某事 |
| 同伴 | `と` | 友達と行く。 | 和某人一起 |

**实用原则**：
  - 想说“东西自己变成某状态”，优先找自动词：電気が消えた。
  - 想说“某人把东西弄成某状态”，优先找他动词：電気を消した。
  - 想强调人为准备好的结果，用 `他动词てある`：資料が置いてある。
  - 想描述自然状态或变化结果，用 `自动词ている`：ドアが閉まっている。

## 6. 时态、体、极性与语气

本章处理“谓语怎样表示时间、状态、完成、否定、推量和说话态度”。日语没有英语式严格时态体系，核心是：**非过去/过去 + 体貌 + 极性 + 语气**。同一个谓语会叠加这些信息。

### 6.1 总览：时态、体、极性、语气

| 层面 | 作用 | 常见形式 | 例子 | 中文理解 |
| ---- | ---- | -------- | ---- | -------- |
| 时态 | 事件相对说话时或主句时的位置 | 非过去、过去 | 行く、行った | 去/去了 |
| 体 | 事件的阶段或状态 | ている、てある、ておく、てしまう | 読んでいる、置いてある | 正在/结果/预先/完了 |
| 极性 | 肯定或否定 | ない、ません、ではない | 行かない、学生ではない | 不去、不是学生 |
| 语气 | 说话人的判断、来源、态度 | だろう、らしい、そうだ、ようだ | 雨らしい、雨のようだ | 好像/据说/大概 |

**谓语叠加顺序的直觉**：

| 形式 | 拆解 | 含义 |
| ---- | ---- | ---- |
| 読む | 基本动作 + 非过去 | 读/要读 |
| 読まない | 动作 + 否定 + 非过去 | 不读 |
| 読んでいる | 动作 + ている | 正在读/一直读 |
| 読んでいた | 动作 + ている + 过去 | 当时正在读/曾一直读 |
| 読んでいない | 动作 + ている + 否定 | 没在读/还没读 |
| 読まなかった | 动作 + 否定 + 过去 | 没读 |
| 読んでしまった | 动作 + 完了/遗憾 + 过去 | 读完了/不小心读了 |
| 読むだろう | 动作 + 推量 | 大概会读 |

**日语判断时间的三个问题**：

| 问题 | 看哪里 | 例子 |
| ---- | ------ | ---- |
| 是现在/未来还是过去？ | 句末谓语的非过去/过去 | 行く / 行った |
| 动作处于哪个阶段？ | ている、てある、ておく、てしまう 等 | 開いている、開けてある |
| 说话人有多确定？ | だろう、かもしれない、はず、らしい 等 | 来るだろう、来るはずだ |

### 6.2 非过去与过去

日语基本时态只有两大类：**非过去**和**过去**。非过去同时覆盖现在、未来、习惯、普遍事实；过去覆盖过去事件、完成、发现、回想和语气缓和。

| 谓语类型 | 非过去肯定 | 过去肯定 | 非过去否定 | 过去否定 |
| -------- | ---------- | -------- | ---------- | -------- |
| 动词 | 行く | 行った | 行かない | 行かなかった |
| い形容词 | 高い | 高かった | 高くない | 高くなかった |
| な形容词 | 静かだ | 静かだった | 静かではない | 静かではなかった |
| 名词谓语 | 学生だ | 学生だった | 学生ではない | 学生ではなかった |
| 礼貌动词 | 行きます | 行きました | 行きません | 行きませんでした |
| 礼貌名词/な形 | 学生です | 学生でした | 学生ではありません | 学生ではありませんでした |

#### 6.2.1 非过去的用法

| 用法 | 例子 | 说明 |
| ---- | ---- | ---- |
| 现在状态 | 彼は東京に住んでいる。 | 状态需常用 `ている` |
| 现在性质 | この店は安い。 | 形容词非过去表示现在性质 |
| 习惯 | 毎朝コーヒーを飲む。 | 经常性动作 |
| 未来 | 明日大阪へ行く。 | 靠时间副词或语境判断未来 |
| 意志/安排 | 来週会議を開く。 | 已决定的未来事项 |
| 普遍事实 | 水は百度で沸騰する。 | 科学、常识、定义 |
| 条件句中的未来 | 雨が降ったら、行かない。 | 从句可用过去形表示条件完成 |

**非过去不是“现在时”**：

| 句子 | 更自然的理解 |
| ---- | ------------ |
| 明日行く。 | 明天去 |
| 毎日行く。 | 每天去 |
| 今行く。 | 现在就去/马上去 |
| 彼はよく行く。 | 他经常去 |

#### 6.2.2 过去形的用法

| 用法 | 例子 | 说明 |
| ---- | ---- | ---- |
| 过去事件 | 昨日映画を見た。 | 昨天看了电影 |
| 完成 | 宿題はもう終わった。 | 已经完成 |
| 状态变化后结果 | 電気が消えた。 | 灯灭了 |
| 发现/想起 | あ、財布があった。 | “原来在这儿” |
| 回想 | あの店は安かった。 | 回忆过去评价 |
| 语气缓和 | よかったら、来てください。 | 比 `よければ` 更柔和 |
| 条件前提 | 終わったら、帰ります。 | 做完之后回去 |

**过去形不一定等于中文“了”**：

| 日语 | 中文 | 说明 |
| ---- | ---- | ---- |
| もう食べた。 | 已经吃了。 | 完成 |
| 昔は静かだった。 | 以前很安静。 | 过去状态 |
| あ、来た。 | 啊，来了。 | 变化/发现 |
| 行ったら分かる。 | 去了就知道。 | 条件完成 |

#### 6.2.3 从句中的相对时态

日语从句里的时态常常相对主句时间，而不只相对现在。

| 句子 | 含义 | 说明 |
| ---- | ---- | ---- |
| 昨日買った本を読んでいる。 | 正在读昨天买的书。 | `買った` 早于现在 |
| 明日使う資料を準備した。 | 准备好了明天要用的资料。 | `使う` 晚于准备动作 |
| 来る人に渡してください。 | 请交给要来的人。 | `来る` 可指未来 |
| 来た人から座ってください。 | 来了的人请先坐。 | `来た` 表到达完成 |

**常见区别**：

| 形式 | 例子 | 差别 |
| ---- | ---- | ---- |
| `Vる前に` | 寝る前に歯を磨く。 | 睡觉之前；前项未发生 |
| `Vた後で` | 寝た後で電話が来た。 | 睡了之后；前项已完成 |
| `Vるとき` | 日本へ行くとき、辞書を買った。 | 去日本之前或途中买 |
| `Vたとき` | 日本へ行ったとき、辞書を買った。 | 到日本后/去过日本时买 |

### 6.3 进行、结果与状态：ている、てある、ておく、てしまう

`ている` 是日语体貌系统的核心。它不只表示“正在”，还表示结果状态、习惯、经历、职业身份等。

#### 6.3.1 ている 的主要用法

| 用法 | 结构 | 例子 | 中文 |
| ---- | ---- | ---- | ---- |
| 正在进行 | 动作动词 + ている | 今、本を読んでいる。 | 正在读书 |
| 结果状态 | 变化动词 + ている | ドアが開いている。 | 门开着 |
| 持续状态 | 状态性动词 + ている | 東京に住んでいる。 | 住在东京 |
| 习惯/反复 | 频率词 + ている | 毎朝走っている。 | 每天跑步 |
| 职业/身份 | 職業を + している | 医者をしている。 | 当医生 |
| 经验记录 | 过去事实仍有关联 | 彼は本を三冊書いている。 | 他写过三本书 |

**同一形式按动词类型变化**：

| 动词 | ている | 常见含义 |
| ---- | ------ | -------- |
| 読む | 読んでいる | 正在读/经常读 |
| 開く | 開いている | 开着 |
| 死ぬ | 死んでいる | 已经死了，不是“正在死” |
| 結婚する | 結婚している | 已婚 |
| 知る | 知っている | 知道 |
| 持つ | 持っている | 拥有/带着 |

**实用判断**：

| 动词类型 | `ている` 常见解释 | 例子 |
| -------- | ---------------- | ---- |
| 可持续动作 | 正在进行 | 食べている、話している |
| 瞬间变化 | 结果状态 | 開いている、壊れている |
| 状态获得 | 持续状态 | 知っている、住んでいる |
| 反复动作 | 习惯 | 通っている、使っている |

#### 6.3.2 てある：人为准备好的结果

`てある` 通常接他动词，表示“有人有意做了某事，结果现在保持着”。重点是**人为安排后的状态**。

| 形式 | 例子 | 说明 |
| ---- | ---- | ---- |
| `Nが 他動詞てある` | 窓が開けてある。 | 窗户被人开好了 |
| `Nに 他動詞てある` | 紙に名前が書いてある。 | 纸上写着名字 |
| `もう Vてある` | 予約はもうしてある。 | 已经预约好了 |
| `ちゃんと Vてある` | 資料はちゃんと準備してある。 | 已妥善准备 |

**ている vs てある**：

| 形式 | 例子 | 重点 |
| ---- | ---- | ---- |
| 自動詞 + ている | ドアが開いている。 | 门处于开着的状态 |
| 他動詞 + てある | ドアが開けてある。 | 有人把门开好了 |
| 他動詞 + ている | 彼がドアを開けている。 | 他正在开门/一直开着门 |

#### 6.3.3 ておく：预先做、暂且保持

`ておく` 表示为了之后而预先做，或暂时让状态保持。口语中常缩约为 `とく`。

| 用法 | 例子 | 说明 |
| ---- | ---- | ---- |
| 预先准备 | 明日のために資料を読んでおく。 | 先读好资料 |
| 暂时放置 | そのままにしておいてください。 | 请先保持原样 |
| 防备 | 念のため、確認しておく。 | 以防万一先确认 |
| 口语缩约 | 予約しとくね。 | `しておく` → `しとく` |

| 易混形式 | 例子 | 差别 |
| -------- | ---- | ---- |
| `Vておく` | 電気をつけておく。 | 先开着灯/保持开着 |
| `Vてある` | 電気がつけてある。 | 灯已经被开好了 |
| `Vている` | 電気がついている。 | 灯亮着 |

#### 6.3.4 てしまう：完成、遗憾、不由得

`てしまう` 一方面表示彻底完成，另一方面常带“遗憾、后悔、出乎意料”。口语缩约：`ちゃう/じゃう`。

| 用法 | 例子 | 说明 |
| ---- | ---- | ---- |
| 完全做完 | 本を全部読んでしまった。 | 全部读完了 |
| 遗憾/后悔 | 財布を忘れてしまった。 | 把钱包忘了 |
| 不由得 | 笑ってしまった。 | 忍不住笑了 |
| 口语缩约 | 食べちゃった。飲んじゃった。 | 轻松口语 |
| 将要做完 | 先に片付けてしまおう。 | 先把它收拾完吧 |

**语气差别**：

| 句子 | 语感 |
| ---- | ---- |
| 宿題をした。 | 做了作业 |
| 宿題をしておいた。 | 预先做好作业 |
| 宿題をしてしまった。 | 已经彻底做完/不小心做了 |
| 宿題をしている。 | 正在做/一直做 |

### 6.4 完成、经验、持续与阶段

日语表示“已经、还没、曾经、一直、刚刚、快要”的方式，通常由时间副词和体貌形式共同承担。

#### 6.4.1 もう、まだ、これから

| 表达 | 例子 | 中文 | 注意 |
| ---- | ---- | ---- | ---- |
| `もう Vた` | もう食べた。 | 已经吃了 | 完成 |
| `もう Vない` | もう行かない。 | 不再去了 | 不再 |
| `まだ Vている` | まだ食べている。 | 还在吃 | 动作持续 |
| `まだ Vていない` | まだ食べていない。 | 还没吃 | 未完成 |
| `まだ Vない` | まだ分からない。 | 还不知道 | 状态未成立 |
| `これから Vる` | これから食べる。 | 接下来要吃 | 将要开始 |

**もう/まだ 的问答**：

| 问句 | 肯定回答 | 否定回答 |
| ---- | -------- | -------- |
| もう昼ご飯を食べましたか。 | はい、もう食べました。 | いいえ、まだ食べていません。 |
| まだ仕事をしていますか。 | はい、まだしています。 | いいえ、もう終わりました。 |

#### 6.4.2 经验：たことがある

`Vたことがある` 表示曾经有过某经验；否定是 `Vたことがない`。

| 形式 | 例子 | 说明 |
| ---- | ---- | ---- |
| `Vたことがある` | 日本へ行ったことがある。 | 去过日本 |
| `Vたことがない` | 納豆を食べたことがない。 | 没吃过纳豆 |
| `一度も Vたことがない` | 一度も会ったことがない。 | 一次也没见过 |
| `何度も Vたことがある` | 何度も聞いたことがある。 | 听过很多次 |

**不能和具体一次性过去时间硬搭**：

| 不自然 | 自然 |
| ------ | ---- |
| 昨日、日本へ行ったことがある。 | 昨日、日本へ行った。 |
| 去年、富士山に登ったことがある。 | 去年、富士山に登った。 |
| 日本へ行ったことがある。 | 日本へ行ったことがある。 |

#### 6.4.3 持续：ずっと、〜続ける、〜ている

| 表达 | 例子 | 说明 |
| ---- | ---- | ---- |
| `ずっと Vている` | ずっと待っている。 | 一直在等 |
| `V続ける` | 三時間歩き続けた。 | 连续做某动作 |
| `Nの間 Vている` | 会議の間、黙っていた。 | 在某期间一直 |
| `〜て以来` | 日本に来て以来、忙しい。 | 自从……以来 |
| `〜てからずっと` | 卒業してからずっと東京にいる。 | 从……后一直 |

| 差别 | 例子 | 说明 |
| ---- | ---- | ---- |
| 状态持续 | 彼はずっと東京に住んでいる。 | 状态延续 |
| 动作连续 | 彼は三時間話し続けた。 | 动作不停 |
| 期间状态 | 子供の間はよく泣いていた。 | 某时期常如此 |

#### 6.4.4 即将、刚刚、差一点

| 表达 | 接续 | 例子 | 含义 |
| ---- | ---- | ---- | ---- |
| `Vるところだ` | 辞书形 | 今から出かけるところだ。 | 正要 |
| `Vているところだ` | ている | 今、調べているところだ。 | 正在 |
| `Vたところだ` | た形 | さっき着いたところだ。 | 刚刚 |
| `Vたばかりだ` | た形 | 今来たばかりだ。 | 刚刚，主观感觉近 |
| `Vそうになる` | ます形词干 | 転びそうになった。 | 差点 |
| `もう少しで Vるところだった` | 辞书形 | もう少しで遅刻するところだった。 | 差点就 |

**ところ vs ばかり**：

| 形式 | 例子 | 重点 |
| ---- | ---- | ---- |
| `着いたところだ` | 今着いたところです。 | 客观“刚到这一刻” |
| `着いたばかりだ` | 一週間前に着いたばかりです。 | 说话人觉得“刚到不久” |

### 6.5 否定、双重否定与部分否定

日语否定不仅是 `ない`，还涉及否定范围、部分否定、强否定、双重否定和礼貌层级。

#### 6.5.1 基本否定形式

| 谓语 | 常体否定 | 礼貌否定 | 过去否定 |
| ---- | -------- | -------- | -------- |
| 动词 | 行かない | 行きません | 行かなかった / 行きませんでした |
| い形容词 | 高くない | 高くないです / 高くありません | 高くなかった |
| な形容词 | 静かではない | 静かではありません | 静かではなかった |
| 名词 | 学生ではない | 学生ではありません | 学生ではなかった |
| ある | ない | ありません | なかった / ありませんでした |

**ではない / じゃない**：

| 形式 | 场合 | 例子 |
| ---- | ---- | ---- |
| `ではない` | 书面、正式、强调 | これは偶然ではない。 |
| `じゃない` | 口语 | これは私のじゃない。 |
| `ではありません` | 礼貌正式 | 問題ではありません。 |
| `じゃありません` | 礼貌口语 | そうじゃありません。 |

#### 6.5.2 否定范围

否定范围决定“不是谁/不是做什么/不是全部/不是一定”。日语常用 `は`、副词位置和句型来标明范围。

| 句子 | 含义 | 焦点 |
| ---- | ---- | ---- |
| 私は行かない。 | 我不去。 | 我这个人 |
| 私が行かない。 | 不是我去/我不去这件事。 | 主语或事实 |
| 東京には行かない。 | 东京不去。 | 对比地点 |
| 毎日は飲まない。 | 不是每天都喝。 | 部分否定 |
| 全部は分からない。 | 不是全部都懂。 | 部分否定 |
| 全然分からない。 | 完全不懂。 | 全否定 |

#### 6.5.3 部分否定与全否定

| 类型 | 日语 | 中文 | 说明 |
| ---- | ---- | ---- | ---- |
| 全否定 | 全部分からない。 | 全都不懂。 | 全部都否定 |
| 部分否定 | 全部は分からない。 | 不全懂。 | 有些懂，有些不懂 |
| 全否定 | みんな来なかった。 | 大家都没来。 | 所有人未到 |
| 部分否定 | みんなは来なかった。 | 不是所有人都来了。 | 可能有人来了 |
| 全否定 | いつも遅れない。 | 总是不迟到。 | 每次都不 |
| 部分否定 | いつもは遅れない。 | 平时不迟到。 | 这次可能不同 |

**常用部分否定搭配**：

| 表达 | 例子 | 含义 |
| ---- | ---- | ---- |
| `必ずしも〜ない` | 必ずしも正しくない。 | 未必正确 |
| `全部は〜ない` | 全部は読めない。 | 不能全部读 |
| `いつも〜とは限らない` | いつも成功するとは限らない。 | 不一定总成功 |
| `誰でも〜わけではない` | 誰でもできるわけではない。 | 不是谁都能 |
| `まったく〜ない` | まったく分からない。 | 完全不 |

#### 6.5.4 双重否定

日语双重否定常表达“不得不、不能不、不是不、并非没有”。语气通常比直接肯定更委婉或更强。

| 形式 | 例子 | 中文 | 语气 |
| ---- | ---- | ---- | ---- |
| `Vないわけにはいかない` | 行かないわけにはいかない。 | 不能不去 | 有义务/压力 |
| `Vずにはいられない` | 笑わずにはいられない。 | 忍不住笑 | 情感自然涌出 |
| `Vないことはない` | 読めないことはない。 | 不是不能读 | 勉强可以 |
| `Aくないことはない` | 高くないことはない。 | 不是不贵 | 有点贵 |
| `Nではないことはない` | 可能ではないことはない。 | 不是不可能 | 可能性低但存在 |
| `〜ないでもない` | 行きたくないでもない。 | 也不是不想去 | 委婉 |

**强度差别**：

| 表达 | 强度 | 例子 |
| ---- | ---- | ---- |
| 行く。 | 中性肯定 | 去 |
| 行かなければならない。 | 必须 | 不去不行 |
| 行かないわけにはいかない。 | 情势逼迫 | 不能不去 |
| 行きたくないことはない。 | 弱肯定 | 也不是不想去 |

#### 6.5.5 负极性表达

有些词常和否定搭配，形成固定强否定。

| 表达 | 例子 | 含义 |
| ---- | ---- | ---- |
| `全然〜ない` | 全然分からない。 | 完全不 |
| `まったく〜ない` | まったく問題ない。 | 完全不 |
| `少しも〜ない` | 少しも寒くない。 | 一点也不 |
| `一度も〜ない` | 一度も行ったことがない。 | 一次也没 |
| `誰も〜ない` | 誰も来なかった。 | 谁都没 |
| `何も〜ない` | 何も食べていない。 | 什么都没 |
| `どこにも〜ない` | どこにもない。 | 哪里都没有 |

### 6.6 推量、传闻、样态与比况

语气形式表达“我根据什么判断、确定度多高、这是听来的还是看起来如此”。这是日语自然表达的关键。

#### 6.6.1 推量与确定度

| 形式 | 接续 | 例子 | 确定度/语气 |
| ---- | ---- | ---- | ----------- |
| `かもしれない` | 普通形 | 明日は雨かもしれない。 | 也许，确定度低 |
| `だろう` | 普通形 | 彼は来るだろう。 | 大概，书面/男性口语感 |
| `でしょう` | 普通形 | 明日は寒いでしょう。 | 礼貌推量/天气预报 |
| `と思う` | 普通形 | 彼は来ると思う。 | 我认为 |
| `はずだ` | 普通形/名词の/な形な | 彼は来るはずだ。 | 按道理应该 |
| `に違いない` | 普通形 | 彼は犯人に違いない。 | 一定是，主观确信 |
| `に決まっている` | 普通形 | 無理に決まっている。 | 肯定，带断定感 |
| `わけがない` | 普通形 | 彼が嘘をつくわけがない。 | 不可能 |

**确定度梯度**：

| 确定度 | 表达 | 例子 |
| ------ | ---- | ---- |
| 低 | かもしれない | 間に合うかもしれない。 |
| 中 | だろう/でしょう | 間に合うでしょう。 |
| 依据推断 | はずだ | 今出れば間に合うはずだ。 |
| 高 | に違いない | 彼は知っているに違いない。 |
| 否定高确定 | わけがない | 彼が知らないわけがない。 |

#### 6.6.2 传闻：そうだ、らしい、という

| 形式 | 接续 | 例子 | 说明 |
| ---- | ---- | ---- | ---- |
| `普通形 + そうだ` | 普通形 | 明日は雨が降るそうだ。 | 听说，据可靠信息 |
| `普通形 + らしい` | 普通形/名词 | 彼は結婚したらしい。 | 据说/看来，带推断 |
| `〜という` | 句子/名词 | 彼は来ないという。 | 据说、叫作 |
| `〜とのことだ` | 句子 | 会議は中止とのことです。 | 正式转述 |
| `〜とか` | 普通形 | 彼は転職したとか。 | 听说，较随意 |

**传闻来源感**：

| 表达 | 来源感 | 例子 |
| ---- | ------ | ---- |
| `そうだ` | 听来的信息，自己不加工 | ニュースによると、雨だそうだ。 |
| `らしい` | 听闻 + 自己推断 | 外が暗い。雨らしい。 |
| `とのことだ` | 正式通知、邮件、公告 | 明日は休みとのことです。 |
| `とか` | 闲谈式传闻 | 彼、会社を辞めたとか。 |

#### 6.6.3 样态：そうだ

样态 `そうだ` 表示“看起来要……/看起来很……”，接续和传闻 `そうだ` 不同。

| 词类 | 接续 | 例子 | 含义 |
| ---- | ---- | ---- | ---- |
| 动词 | ます形词干 + そうだ | 雨が降りそうだ。 | 快要下雨 |
| い形容词 | 去い + そうだ | おいしそうだ。 | 看起来好吃 |
| な形容词 | 词干 + そうだ | 元気そうだ。 | 看起来精神 |
| 否定样态 | なさそうだ | 問題はなさそうだ。 | 看起来没有 |
| いい | よさそうだ | いい店 → よさそうな店 | 看起来不错 |

| 形式 | 例子 | 差别 |
| ---- | ---- | ---- |
| 雨が降るそうだ。 | 听说会下雨。 | 传闻 |
| 雨が降りそうだ。 | 看起来快下雨。 | 样态 |
| おいしいそうだ。 | 听说很好吃。 | 传闻，接普通形 |
| おいしそうだ。 | 看起来好吃。 | 样态，去 `い` |

#### 6.6.4 比况与推测：ようだ、みたいだ、らしい

| 形式 | 接续 | 例子 | 语感 |
| ---- | ---- | ---- | ---- |
| `ようだ` | 普通形/名词の/な形な | 夢のようだ。熱があるようだ。 | 书面、较客观 |
| `みたいだ` | 普通形/名词 | 夢みたいだ。雨みたいだ。 | 口语 |
| `らしい` | 普通形/名词 | 彼は来ないらしい。学生らしい。 | 传闻/典型性 |
| `っぽい` | 名词/ます形词干/形容词词干 | 子供っぽい。忘れっぽい。 | 倾向、带口语评价 |

**ようだ / みたいだ / らしい 的区别**：

| 形式 | 例子 | 含义 |
| ---- | ---- | ---- |
| 彼は病気のようだ。 | 看证据推测他好像病了。 | 观察推测 |
| 彼は病気みたいだ。 | 他好像病了。 | 口语推测 |
| 彼は病気らしい。 | 听说/看来他病了。 | 传闻或迹象 |
| 彼は学生らしい。 | 他很有学生样。 | 典型性 |
| 彼は学生のようだ。 | 他像学生。 | 比喻/推测 |

#### 6.6.5 必然、当然与解释

| 形式 | 例子 | 说明 |
| ---- | ---- | ---- |
| `わけだ` | 彼は十年日本に住んでいる。日本語が上手なわけだ。 | 难怪、也就是说 |
| `わけではない` | 嫌いなわけではない。 | 并不是…… |
| `わけがない` | 彼が来るわけがない。 | 不可能 |
| `はずだ` | もう着くはずだ。 | 按理应该 |
| `べきだ` | 約束は守るべきだ。 | 应该，规范判断 |
| `ものだ` | 人は変わるものだ。 | 一般道理/感慨 |

### 6.7 确认、强调与感叹

句末语气和副词决定说话的“姿态”：确认、征求同意、强调、提醒、感叹、意外等。它们通常不改变事实内容，但强烈影响自然度。

#### 6.7.1 确认与征求同意

| 形式 | 例子 | 功能 | 语气 |
| ---- | ---- | ---- | ---- |
| `ね` | 今日は寒いですね。 | 寻求共感/确认 | 柔和 |
| `よね` | 明日、会議ですよね。 | 我认为如此，请确认 | 常用 |
| `でしょう？` | いいでしょう？ | 你也这么觉得吧 | 可柔可强 |
| `だろう？` | 分かるだろう？ | 你明白吧 | 较强 |
| `じゃない？` | これ、便利じゃない？ | 不是挺方便吗 | 口语确认 |
| `ではありませんか` | 重要ではありませんか。 | 正式反问确认 | 书面/演讲 |

**实用选择**：

| 场景 | 推荐 | 例子 |
| ---- | ---- | ---- |
| 普通寒暄 | `ね` | 暑いですね。 |
| 确认日程 | `よね` | 集合は三時ですよね。 |
| 委婉提出看法 | `じゃないでしょうか` | 少し早いんじゃないでしょうか。 |
| 强势确认 | `だろう` | 当然だろう。 |

#### 6.7.2 强调、限定与对比

| 形式 | 例子 | 说明 |
| ---- | ---- | ---- |
| `こそ` | 今度こそ成功したい。 | 正是、才 |
| `さえ` | 名前さえ知らない。 | 连……都 |
| `まで` | 子供まで知っている。 | 连……也 |
| `だけ` | 水だけ飲んだ。 | 只 |
| `しか〜ない` | 水しか飲まなかった。 | 只，必须接否定 |
| `ばかり` | 甘いものばかり食べる。 | 光是、总是 |
| `は` | 東京には行った。 | 对比 |
| `も` | 東京にも行った。 | 也/连 |

**だけ vs しかない**：

| 形式 | 例子 | 语气 |
| ---- | ---- | ---- |
| `だけ` | 千円だけある。 | 中性：只有一千日元 |
| `しか〜ない` | 千円しかない。 | 不足感：才一千日元 |
| `だけしか〜ない` | 千円だけしかない。 | 强调不足 |

#### 6.7.3 感叹与意外

| 形式 | 例子 | 语气 |
| ---- | ---- | ---- |
| `なあ` | きれいだなあ。 | 感叹、自言自语 |
| `ことだ` | なんと美しいことだ。 | 书面感叹 |
| `ものだ` | 時間がたつのは早いものだ。 | 普遍感慨 |
| `なんて` | なんて便利なんだ。 | 多么/竟然 |
| `とは` | 彼が来るとは思わなかった。 | 没想到 |
| `なんか/なんて` | 私なんか無理です。 | 自谦、轻视、举例 |

| 句子 | 中文 | 说明 |
| ---- | ---- | ---- |
| まさか彼が来るとは。 | 没想到他会来。 | 意外 |
| なんて難しい問題だ。 | 多么难的问题。 | 感叹 |
| 時間が過ぎるのは早いものだ。 | 时间过得真快啊。 | 感慨 |
| 私なんてまだまだです。 | 我还差得远呢。 | 自谦 |

#### 6.7.4 语气副词速查

| 副词 | 常搭配 | 例子 | 作用 |
| ---- | ------ | ---- | ---- |
| たぶん | だろう/と思う | たぶん大丈夫だろう。 | 大概 |
| きっと | はず/だろう | きっと成功する。 | 一定会，期待较强 |
| ぜひ | てください/たい | ぜひ来てください。 | 务必 |
| どうしても | たい/ない/必要 | どうしても行きたい。 | 无论如何 |
| まさか | とは/ないだろう | まさか失敗するとは。 | 没想到/难道 |
| どうやら | ようだ/らしい | どうやら雨らしい。 | 看来 |
| なんとなく | 気がする | なんとなく分かる。 | 总觉得 |
| もちろん | 肯定表达 | もちろん行きます。 | 当然 |
| べつに | 否定 | べつに嫌いではない。 | 并不特别 |
| あまり | 否定 | あまり高くない。 | 不太 |

### 6.8 常见错误速查

| 错误 | 正确 | 原因 |
| ---- | ---- | ---- |
| 明日行った。 | 明日行く。 | 未来用非过去 |
| 今ご飯を食べる。 | 今ご飯を食べている。 | 正在进行通常用 `ている` |
| ドアが開けている。 | ドアが開いている。 | 自动词结果状态用 `開いている` |
| ドアが開いてある。 | ドアが開いている / ドアが開けてある。 | 自动词结果用 `ている`；人为准备结果用他动词 `てある` |
| もう食べていない。 | まだ食べていない。 | “还没吃”用 `まだ〜ていない` |
| 全部分からない。 | 全部は分からない。 | “不全懂”要用部分否定 |
| おいしいそうだ。 | おいしそうだ。 | 样态 `そうだ`：い形去 `い` |
| 雨が降りそうだ。 | 雨が降るそうだ。 | 若表示“听说下雨”用传闻接普通形 |
| 彼は来るはずですか。 | 彼は来るはずです。/来るでしょうか。 | `はず` 是根据判断，不常直接询问对方 |
| 日本へ行ったことがある、昨日。 | 昨日日本へ行った。 | 经验句不搭具体一次性过去时间 |


## 7. 复句、从句与接续

复句的核心是把一个句子压缩成句子成分，或把两个事件连接起来。日语复句尤其依赖**普通形、名词化、连体修饰、接续助词和接续词**。本章按“怎么接、表示什么关系、语气差别”来整理。

### 7.1 复句总览：从句怎样进入主句

| 类型 | 结构 | 例子 | 作用 |
| ---- | ---- | ---- | ---- |
| 名词化 | `普通形 + こと/の` | 日本語を勉強することは楽しい。 | 把句子变成名词成分 |
| 引用 | `普通形 + と` | 彼は来ると言った。 | 转述话语、想法、内容 |
| 连体修饰 | `从句 + 名词` | 昨日買った本 | 用句子修饰名词 |
| 时间从句 | `Vる前に / Vた後で` | 食べる前に手を洗う。 | 表时间关系 |
| 原因从句 | `から / ので / ため` | 雨だから行かない。 | 表原因、理由 |
| 条件从句 | `たら / ば / と / なら` | 雨が降ったら行かない。 | 表假设、条件 |
| 逆接从句 | `が / けど / のに / ても` | 高いが、買った。 | 表转折、让步 |
| 目的从句 | `ように / ために` | 忘れないようにメモする。 | 表目的 |
| 结果从句 | `結果 / 末に / て` | 練習して、上手になった。 | 表结果、经过 |

**接续形的快速判断**：

| 接续 | 常接什么 | 典型句型 |
| ---- | -------- | -------- |
| 普通形 | 句子整体 | と思う、こと、の、から、けど、なら |
| 辞书形 | 未发生、目的、前项 | 前に、ために、ように、べき |
| た形 | 完成后、经验、条件 | 後で、ことがある、たら、ところ |
| て形 | 顺接、原因、并列、方式 | 食べて寝る、疲れて寝た |
| ます形词干 | 目的移动、持续复合 | 買いに行く、読み続ける |
| 名词/な形 + な | 连体或推测接续 | 静かな町、静かなようだ |
| 名词 + の | 所属、说明、比况 | 先生の本、夢のようだ |

### 7.2 名词化：こと、の、もの

名词化把一个动作、状态或事实变成名词成分，使它可以做主语、宾语、补语或判断内容。

#### 7.2.1 こと 与 の

| 形式 | 主要功能 | 例子 | 说明 |
| ---- | -------- | ---- | ---- |
| `Vること` | 抽象事情、规则、能力、决定 | 日本語を話すことは難しい。 | 偏概念化 |
| `Vるの` | 具体动作、眼前感受、感知对象 | 彼が走るのを見た。 | 偏具体、生动 |
| `Vたこと` | 经验、事实 | 日本へ行ったことがある。 | “做过某事” |
| `普通形 + ことだ` | 建议、感叹、解释 | 休みたいなら早く寝ることだ。 | 句型化 |
| `普通形 + のだ/んだ` | 说明、解释、强调 | 電車が遅れたんです。 | 解释背景 |

**こと/の 的选择**：

| 场景 | 多用 | 例子 |
| ---- | ---- | ---- |
| 主语是抽象行为 | こと/の 都可 | 勉強すること/のは大切だ。 |
| 感知动词对象 | の | 彼が出るのを見た。 |
| 阻止、等待、帮忙等具体动作 | の | 行くのをやめた。 |
| 能力、可能、决定、约定 | こと | 泳ぐことができる。行くことにした。 |
| 经验 | こと | 富士山に登ったことがある。 |
| 说明语气 | のだ/んだ | 今日は休みなんです。 |

| 常见搭配 | 例子 | 含义 |
| -------- | ---- | ---- |
| `Vることができる` | 漢字を読むことができる。 | 能够 |
| `Vることにする` | 毎日走ることにした。 | 自己决定 |
| `Vることになる` | 転勤することになった。 | 外部决定/结果 |
| `Vることがある` | 時々遅れることがある。 | 有时会 |
| `Vたことがある` | 見たことがある。 | 曾经 |
| `Vるのを忘れる` | 電気を消すのを忘れた。 | 忘记做 |
| `Vるのが好き` | 歩くのが好きだ。 | 喜欢做 |

#### 7.2.2 もの、ところ、わけ

| 形式 | 例子 | 功能 |
| ---- | ---- | ---- |
| `もの` | 甘いものが好きだ。 | 东西、物、抽象对象 |
| `ものだ` | 子供はよく泣くものだ。 | 一般道理、感慨 |
| `ものではない` | 人を笑うものではない。 | 不应该 |
| `ところ` | 今帰るところだ。 | 阶段、地方、点 |
| `ところを` | 忙しいところをすみません。 | 正当……时候 |
| `わけ` | 遅れたわけを説明する。 | 理由、道理 |
| `わけだ` | だから高いわけだ。 | 难怪、也就是说 |
| `わけではない` | 嫌いなわけではない。 | 并不是 |

**名词化常见错误**：

| 错误 | 正确 | 原因 |
| ---- | ---- | ---- |
| 彼が走ることを見た。 | 彼が走るのを見た。 | 感知对象多用 `の` |
| 日本へ行くのができる。 | 日本へ行くことができる。 | 能力句型固定用 `こと` |
| 電気を消すことを忘れた。 | 電気を消すのを忘れた。 | 忘记具体动作常用 `の` |
| 休みなのことです。 | 休みだということです。 | 转述内容用 `ということ` |

### 7.3 引用与转述

引用把话语、想法、判断、命令、疑问等嵌入主句。核心标记是 `と`，口语中常用 `って`。

#### 7.3.1 と、って、という

| 形式 | 用法 | 例子 | 语气 |
| ---- | ---- | ---- | ---- |
| `普通形 + と言う` | 直接/间接引用 | 彼は行くと言った。 | 中性 |
| `普通形 + って言う` | 口语引用 | 彼、行くって言ってた。 | 口语 |
| `句子 + という + 名词` | 叫作/内容说明 | 田中さんという人 | 说明 |
| `句子 + とのこと` | 正式转述 | 明日は休みとのことです。 | 正式 |
| `句子 + そうだ` | 传闻 | 雨が降るそうだ。 | 听说 |
| `句子 + らしい` | 传闻/推断 | 彼は来ないらしい。 | 听闻+判断 |

| 被引用内容 | 例子 | 说明 |
| ---------- | ---- | ---- |
| 陈述 | 彼は忙しいと言った。 | “他说很忙” |
| 意志 | 彼は行くと言った。 | “他说要去” |
| 命令 | 先生は来いと言った。 | 直接命令内容 |
| 请求 | 彼は待ってくださいと言った。 | 礼貌请求原文 |
| 疑问 | 彼は何時に来るか聞いた。 | 疑问嵌入不用 `と` 作唯一选择 |

#### 7.3.2 と思う、考える、感じる

| 形式 | 例子 | 说明 |
| ---- | ---- | ---- |
| `普通形 + と思う` | 彼は来ると思う。 | 我认为 |
| `普通形 + と思っている` | 彼は正しいと思っている。 | 持续持有想法 |
| `意志形 + と思う` | 明日行こうと思う。 | 打算 |
| `普通形 + と考える` | 必要だと考える。 | 较正式 |
| `普通形 + と感じる` | 少し変だと感じる。 | 感觉 |

**と思う 的人称限制**：

| 句子 | 自然度 | 说明 |
| ---- | ------ | ---- |
| 私は行きたいと思う。 | 自然 | 自己想法 |
| 彼は行きたいと思っている。 | 自然 | 别人的想法用持续状态 |
| 彼は行きたいと思う。 | 不自然/需语境 | 像“我觉得他想去”时应改成 `と思っていると思う` 等 |

#### 7.3.3 疑问从句

| 形式 | 例子 | 含义 |
| ---- | ---- | ---- |
| `疑问词 + か` | 何を買うか分からない。 | 不知道买什么 |
| `是否 + かどうか` | 行くかどうか迷っている。 | 是否去 |
| `普通形 + か` | 彼が来るか知りたい。 | 想知道他来不来 |
| `疑问词 + でも` | 何でも聞いてください。 | 什么都 |
| `疑问词 + か` | 誰か来た。 | 有人 |

| 对比 | 例子 | 说明 |
| ---- | ---- | ---- |
| `誰が来るか分からない` | 不知道谁来 | 疑问从句 |
| `誰かが来た` | 有人来了 | 不定指 |
| `誰でも来られる` | 谁都能来 | 任指 |
| `誰も来ない` | 谁都不来 | 否定任指 |

### 7.4 连体修饰从句

日语可以直接用一个普通形句子修饰名词，相当于中文“……的”。被修饰名词和从句内部的关系靠语义判断，不加关系代词。

| 结构 | 例子 | 中文 |
| ---- | ---- | ---- |
| `Vる + 名词` | 明日読む本 | 明天要读的书 |
| `Vた + 名词` | 昨日買った本 | 昨天买的书 |
| `Vている + 名词` | そこで待っている人 | 在那里等的人 |
| `い形 + 名词` | 高い店 | 贵的店 |
| `な形 + な + 名词` | 静かな町 | 安静的城市 |
| `名词 + の + 名词` | 日本語の本 | 日语书 |

#### 7.4.1 从句内部的格关系

| 关系 | 例子 | 说明 |
| ---- | ---- | ---- |
| 主语 | 昨日来た人 | 来的人 |
| 宾语 | 私が読んだ本 | 我读的书 |
| 地点 | 私が住んでいる町 | 我住的城市 |
| 时间 | 彼が生まれた年 | 他出生的年份 |
| 工具 | ご飯を食べる箸 | 吃饭用的筷子 |
| 内容 | 彼が来るという話 | 他说要来的消息 |

**从句主语常用 が**：

| 自然 | 不自然/特殊 | 说明 |
| ---- | ---------- | ---- |
| 私が買った本 | 私は買った本 | 连体从句内部主语多用 `が` |
| 先生が書いた本 | 先生は書いた本 | `は` 会造成对比或断裂感 |

#### 7.4.2 内外关系与歧义

| 句子 | 可能含义 | 说明 |
| ---- | -------- | ---- |
| 彼が撮った写真 | 他拍的照片 | 他是拍摄者 |
| 彼を撮った写真 | 拍他的照片 | 他是被拍对象 |
| 母が作った料理 | 妈妈做的菜 | 主体关系 |
| 母に作った料理 | 给妈妈做的菜 | 受益/对象关系 |
| 日本で買った時計 | 在日本买的表 | 地点 |

**实用原则**：
  - 从句里缺哪个成分，后面的名词通常就补那个成分。
  - 如果关系容易误解，用助词保留清楚：彼を撮った写真、母に作った料理。
  - 连体从句不用中文式“的”的固定位置，而是整个从句直接放在名词前。

### 7.5 时间与顺序

时间从句说明两个事件的先后、同时、期间、起点和终点。关键在于前项是否完成、动作是否持续。

#### 7.5.1 前后关系

| 形式 | 接续 | 例子 | 含义 |
| ---- | ---- | ---- | ---- |
| `Vる前に` | 辞书形 | 寝る前に歯を磨く。 | 做……之前 |
| `Vた後で` | た形 | 食べた後で薬を飲む。 | 做……之后 |
| `Vてから` | て形 | 宿題をしてから寝る。 | 做完后再 |
| `Nの前に/後で` | 名词 | 会議の前に確認する。 | ……之前/之后 |
| `V次第` | ます形词干 | 終わり次第連絡します。 | 一……就，正式 |
| `Vたところ` | た形 | 着いたところ電話が来た。 | 刚……时 |

**後で vs てから**：

| 形式 | 例子 | 差别 |
| ---- | ---- | ---- |
| `食べた後で寝る` | 吃完后睡 | 单纯时间顺序 |
| `食べてから寝る` | 吃了再睡 | 强调先做前项，再做后项 |
| `説明を聞いてから質問する` | 听完说明再提问 | 常用于步骤 |

#### 7.5.2 同时与期间

| 形式 | 接续 | 例子 | 含义 |
| ---- | ---- | ---- | ---- |
| `Vながら` | ます形词干 | 音楽を聞きながら勉強する。 | 一边……一边 |
| `Vている間` | ている | 留守の間、猫を預かった。 | 在……期间 |
| `Vる間に` | 辞书形 | 寝ている間に雨が止んだ。 | 在期间发生变化 |
| `N中` | 名词 | 会議中、電話しないでください。 | 正在……中 |
| `V最中に` | ている/名词の | 食事の最中に電話が来た。 | 正当……时 |
| `Vところに/で/を` | 普通形 | 出かけるところに客が来た。 | 正要/正在/刚……时 |

| 形式 | 例子 | 说明 |
| ---- | ---- | ---- |
| `間` | 日本にいる間、働いた。 | 整个期间 |
| `間に` | 日本にいる間に、友達ができた。 | 期间内某时发生 |
| `ながら` | 歩きながら話す。 | 同一主体同时动作 |

#### 7.5.3 起点、终点与持续

| 形式 | 例子 | 含义 |
| ---- | ---- | ---- |
| `Vて以来` | 卒業して以来、会っていない。 | 自从……以来 |
| `Vてからずっと` | 来日してからずっと東京にいる。 | 从……后一直 |
| `NからNまで` | 月曜から金曜まで働く。 | 从……到…… |
| `Vるまで` | 終わるまで待つ。 | 直到……为止 |
| `Vないうちに` | 忘れないうちにメモする。 | 趁还没…… |
| `Vるうちに` | 話しているうちに分かった。 | 在……过程中 |

### 7.6 原因、理由与依据

原因表达分层很细：`から` 主观直接，`ので` 客观柔和，`ため` 书面正式，`おかげで` 偏好结果，`せいで` 偏坏结果。

| 形式 | 接续 | 例子 | 语气 |
| ---- | ---- | ---- | ---- |
| `から` | 普通形 | 雨だから行かない。 | 主观、直接 |
| `ので` | 普通形/な形な/名词な | 雨なので行きません。 | 客观、柔和 |
| `ために` | 普通形/名词の | 雨のため、延期になった。 | 正式、书面 |
| `おかげで` | 普通形/名词の | 先生のおかげで合格した。 | 多亏，正面 |
| `せいで` | 普通形/名词の | 雨のせいで遅れた。 | 都怪，负面 |
| `によって` | 名词 | 台風によって被害が出た。 | 客观原因 |
| `以上` | 普通形 | 約束した以上、守るべきだ。 | 既然 |
| `からには` | 普通形 | やるからには勝ちたい。 | 既然要做 |

**から vs ので**：

| 场景 | 推荐 | 例子 |
| ---- | ---- | ---- |
| 朋友间解释 | `から` | 疲れたから帰る。 |
| 正式说明原因 | `ので` | 体調が悪いので休みます。 |
| 请求前置理由 | `ので` | 少し寒いので、窓を閉めてもいいですか。 |
| 强烈主张 | `から` | 危ないからやめなさい。 |

**原因表达常见搭配**：

| 形式 | 例子 | 注意 |
| ---- | ---- | ---- |
| `なぜなら〜からだ` | 行かない。なぜなら忙しいからだ。 | 书面论证 |
| `どうしてかというと〜からだ` | 理由は、時間がないからだ。 | 解释原因 |
| `〜ばかりに` | 一言言ったばかりに怒られた。 | 只因为……导致坏结果 |
| `〜だけに` | 有名なだけに、人が多い。 | 正因为 |

### 7.7 条件与假设

日语条件主要有 `たら`、`ば`、`と`、`なら`。它们都可译为“如果”，但时间顺序、自然规律、话题前提和后项限制不同。

#### 7.7.1 四大条件

| 形式 | 接续 | 例子 | 核心语感 |
| ---- | ---- | ---- | -------- |
| `たら` | た形 | 雨が降ったら行かない。 | 前项实现后，后项发生 |
| `ば` | 条件形 | 安ければ買う。 | 条件成立则 |
| `と` | 辞书形/ない形 | 春になると花が咲く。 | 自然结果、恒常 |
| `なら` | 普通形/名词 | 東京へ行くなら新幹線が便利だ。 | 既然/如果说到 |

| 形式 | 后项能否接意志/请求 | 例子 | 说明 |
| ---- | ------------------ | ---- | ---- |
| `たら` | 可以 | 暇だったら来てください。 | 最自由 |
| `ば` | 有限制 | 時間があれば来てください。 | 前项是状态时较自然 |
| `と` | 通常不接意志请求 | 春になると花が咲く。 | 表自动结果 |
| `なら` | 可以 | 行くなら連絡してください。 | 以对方话题为前提 |

#### 7.7.2 たら、ば、と、なら 的区别

| 句子 | 含义 | 适用 |
| ---- | ---- | ---- |
| 家に帰ったら電話します。 | 回到家后打电话。 | 时间先后 |
| 家に帰れば安心できる。 | 只要回家就能安心。 | 条件成立 |
| 家に帰ると、猫がいた。 | 一回家，猫在。 | 发现/自然衔接 |
| 家に帰るなら、これを持って帰って。 | 如果你要回家，把这个带回去。 | 承接对方计划 |

| 错误倾向 | 更自然 | 原因 |
| -------- | ------ | ---- |
| 春になったら花が咲く。 | 春になると花が咲く。 | 自然规律用 `と` 更典型 |
| 東京へ行ったら新幹線が便利だ。 | 東京へ行くなら新幹線が便利だ。 | 建议基于“要去东京”这个前提 |
| 安いと買ってください。 | 安かったら買ってください。 | 请求后项不用 `と` |

#### 7.7.3 让步条件

| 形式 | 例子 | 含义 |
| ---- | ---- | ---- |
| `Vても` | 雨が降っても行く。 | 即使 |
| `いくら〜ても` | いくら説明しても分からない。 | 无论怎么 |
| `どんなに〜ても` | どんなに忙しくても寝る。 | 无论多么 |
| `たとえ〜ても` | たとえ失敗しても続ける。 | 即使，书面强调 |
| `〜にしても` | 行くにしても準備が必要だ。 | 即便要…… |

### 7.8 逆接、让步与对比

逆接表达“虽然/但是/可是/明明”。要区分单纯转折、出乎预期、遗憾责备和正式对比。

| 形式 | 接续 | 例子 | 语气 |
| ---- | ---- | ---- | ---- |
| `が` | 普通形 | 高いが、質はいい。 | 书面、中性 |
| `けど/けれど` | 普通形 | 高いけど、買った。 | 口语、柔和 |
| `しかし` | 句首 | 便利だ。しかし高い。 | 正式转折 |
| `でも` | 句首 | 行きたい。でも時間がない。 | 口语转折 |
| `のに` | 普通形/な形な/名词な | 雨なのに出かけた。 | 明明……却 |
| `ながら` | ます形词干/名词 | 知っていながら黙っていた。 | 虽然，书面 |
| `ものの` | 普通形 | 買ったものの、使っていない。 | 虽然……但是 |
| `とはいえ` | 普通形 | 安いとはいえ、品質は悪くない。 | 虽说 |

**が/けど vs のに**：

| 形式 | 例子 | 差别 |
| ---- | ---- | ---- |
| 高いが、買った。 | 虽然贵但买了。 | 中性转折 |
| 高いけど、買った。 | 贵是贵，但买了。 | 口语 |
| 高いのに、買った。 | 明明很贵却买了。 | 含意外/责备/不满 |

**句首连接词**：

| 连接词 | 例子 | 语体 |
| ------ | ---- | ---- |
| しかし | 便利です。しかし、費用が高いです。 | 正式 |
| ただ | 便利です。ただ、少し高いです。 | 补充限制 |
| でも | 行きたい。でも、忙しい。 | 口语 |
| ところが | 出発した。ところが、雨が降ってきた。 | 出乎预料 |
| それでも | 失敗した。それでも続けた。 | 即便如此 |

### 7.9 目的、手段与结果

目的说明“为了什么”，手段说明“通过什么方式”，结果说明“导致什么”。`ために` 和 `ように` 是重点。

#### 7.9.1 目的：ために、ように、に

| 形式 | 接续 | 例子 | 适用 |
| ---- | ---- | ---- | ---- |
| `Vるために` | 辞书形 | 留学するために貯金する。 | 主体可控目的 |
| `Nのために` | 名词の | 家族のために働く。 | 为了某人/某事 |
| `Vるように` | 辞书形/ない形 | 忘れないようにメモする。 | 状态达成、避免、能力 |
| `ます形 + に行く` | ます形词干 | 本を買いに行く。 | 移动目的 |
| `Vようと思う` | 意志形 | 明日行こうと思う。 | 打算 |

**ために vs ように**：

| 形式 | 例子 | 说明 |
| ---- | ---- | ---- |
| 合格するために勉強する。 | 为了合格而学习。 | 可控目的 |
| 合格できるように勉強する。 | 为了能合格而学习。 | 能力/状态达成 |
| 忘れないようにメモする。 | 为了不忘而记下来。 | 否定目的常用 |
| 家を買うために貯金する。 | 为了买房存钱。 | 直接目的 |

#### 7.9.2 手段、方式与伴随

| 形式 | 例子 | 含义 |
| ---- | ---- | ---- |
| `で` | 電車で行く。 | 用、通过 |
| `によって` | 方法によって結果が違う。 | 根据/通过 |
| `を通じて` | 仕事を通じて学んだ。 | 通过某过程 |
| `を通して` | 友人を通して知り合った。 | 通过中介 |
| `として` | 代表として参加する。 | 作为 |
| `とともに` | 家族とともに暮らす。 | 和……一起/同时 |
| `なしで` | 予約なしで入れる。 | 没有……也 |

#### 7.9.3 结果与变化

| 形式 | 例子 | 说明 |
| ---- | ---- | ---- |
| `Vて、結果` | 転んでけがをした。 | 顺接导致结果 |
| `その結果` | 努力した。その結果、合格した。 | 书面结果 |
| `〜ため` | 雨のため中止になった。 | 原因导致 |
| `〜あげく` | 悩んだあげく、断った。 | 经过一番后，多负面 |
| `〜末に` | 相談した末に決めた。 | 经过……之后 |
| `〜結果` | 調査した結果、分かった。 | 调查/研究后结果 |
| `〜ようになる` | 日本語が話せるようになった。 | 变得能够/习惯 |

### 7.10 并列、递进与添加

并列和递进用来排列多个信息。日语要注意每种形式的语体、是否穷尽、是否带评价。

| 形式 | 接续 | 例子 | 说明 |
| ---- | ---- | ---- | ---- |
| `Aて、B` | て形 | 安くて便利だ。 | 并列、原因、顺接 |
| `Aし、Bし` | 普通形 | 安いし、便利だし、よく使う。 | 列举理由 |
| `AたりBたりする` | た形 | 読んだり書いたりする。 | 举例动作，不穷尽 |
| `AやBなど` | 名词 | 本や雑誌など | 名词举例 |
| `AとかBとか` | 名词/句子 | 映画とか音楽とかが好き。 | 口语举例 |
| `AもBも` | 名词 | 父も母も教師だ。 | 两者都 |
| `AだけでなくBも` | 普通形/名词 | 安いだけでなく、質もいい。 | 不仅而且 |
| `AばかりかBも` | 普通形/名词 | 遅刻したばかりか、宿題も忘れた。 | 不但，递进较强 |
| `Aに加えてB` | 名词 | 雨に加えて風も強い。 | 加上 |

**て形并列的限制**：

| 自然 | 不自然 | 原因 |
| ---- | ------ | ---- |
| この部屋は広くて明るい。 | この部屋は広くて、昨日掃除した。 | `て` 并列需关系自然 |
| 彼は親切で、仕事も早い。 | 彼は学生で、走る。 | 名词/动作并列需改结构 |

**たり 的重点**：

| 例子 | 说明 |
| ---- | ---- |
| 休みの日は本を読んだり映画を見たりします。 | 举例说明休息日活动 |
| 昨日は本を読んだりしました。 | 只举一个例子也可，暗示还有别的 |
| 行ったり来たりする。 | 来来回回，固定往复 |

### 7.11 比较、程度与范围

比较结构处理“比、最、和……一样、越……越、在……中”。本节补充第 8 章功能句型前的复句连接基础。

| 形式 | 例子 | 含义 |
| ---- | ---- | ---- |
| `AはBより〜` | 東京は大阪より大きい。 | A比B更 |
| `BよりAのほうが〜` | 大阪より東京のほうが大きい。 | 比起B，A更 |
| `AとBとどちらが〜` | コーヒーとお茶とどちらが好きですか。 | 二选一比较 |
| `AはBほど〜ない` | 今日は昨日ほど寒くない。 | 不如B那么 |
| `Aと同じくらい〜` | 彼は私と同じくらい背が高い。 | 和……一样 |
| `Nの中で一番〜` | 日本料理の中で寿司が一番好きだ。 | 在……中最 |
| `〜ば〜ほど` | 読めば読むほど面白い。 | 越……越 |
| `〜くらい/ほど` | 歩けないほど疲れた。 | 到……程度 |

**より/ほど 的方向**：

| 句子 | 中文 | 说明 |
| ---- | ---- | ---- |
| AはBより高い。 | A比B贵。 | `より` 后是比较基准 |
| AはBほど高くない。 | A没有B那么贵。 | `ほど〜ない` 是否定比较 |
| BよりAのほうが高い。 | 比起B，A更贵。 | `ほう` 标更高一方 |

**程度副词**：

| 副词 | 例子 | 说明 |
| ---- | ---- | ---- |
| とても | とても寒い。 | 很 |
| かなり | かなり難しい。 | 相当 |
| ずいぶん | ずいぶん変わった。 | 变化大，带感受 |
| 少し | 少し高い。 | 稍微 |
| あまり〜ない | あまり高くない。 | 不太 |
| それほど〜ない | それほど難しくない。 | 没那么 |
| ほとんど〜ない | ほとんど分からない。 | 几乎不 |

### 7.12 常见错误速查

| 错误 | 正确 | 原因 |
| ---- | ---- | ---- |
| 彼が走ることを見た。 | 彼が走るのを見た。 | 感知对象用 `の` 更自然 |
| 泳ぐのができます。 | 泳ぐことができます。 | 能力句型固定用 `こと` |
| 私は買った本 | 私が買った本 | 连体从句内部主语多用 `が` |
| 雨だから、窓を閉めてもいいですか。 | 雨なので、窓を閉めてもいいですか。 | 请求前理由用 `ので` 更柔和 |
| 春になったら花が咲く。 | 春になると花が咲く。 | 自然规律用 `と` |
| 東京へ行ったら新幹線が便利だ。 | 東京へ行くなら新幹線が便利だ。 | 建议基于前提用 `なら` |
| 安いと買ってください。 | 安かったら買ってください。 | `と` 后项一般不接请求 |
| 雨でも、行かなかった。 | 雨だったので、行かなかった。 | `ても` 是让步，“即使也” |
| 合格できるために勉強する。 | 合格できるように勉強する。 | 能力达成目的用 `ように` |
| 本を読むとか映画を見る。 | 本を読んだり映画を見たりする。 | 动作举例常用 `たり` |

| 舌を巻く（したをまく）             | 佩服、惊叹                    | 形容对某人的能力、技巧感到非常惊讶或钦佩，通常用于表达佩服的语境               | 彼の日本語の能力には**舌を巻く**。<br>（他的日语能力让我佩服得五体投地。）    |
| 足を洗う（あしをあらう）           | 改邪归正、金盆洗手             | 表示不再做坏事，脱离某种不好的环境或不良习惯                                 | 彼は昔、悪いことをしていたが、今は**足を洗って**真面目に働いている。<br>（他以前做过坏事，但现在改邪归正了。） |
| 猫をかぶる（ねこをかぶる）         | 装乖、伪装善良                | 形容在别人面前装得很老实、乖巧，隐藏自己的真实个性                              | 彼女は普段は**猫をかぶって**いるが、実はかなりおしゃべりだ。<br>（她平时装得很乖，其实很爱说话。） |
| 竹を割ったよう（たけをわったよう） | 性格爽快、光明磊落             | 形容某人的性格直爽、开朗、不做作，常用来形容个性分明、坦率的人                   | 彼は**竹を割ったような**性格で、誰にでも公平だ。<br>（他性格直爽，对谁都很公平。） |
| 目を盗む（めをぬすむ）             | 背着某人、偷偷地               | 表示背着某人、瞒着他人做某事，常用于描述偷偷进行的行为                           | 彼女は親の**目を盗んで**夜中にゲームをした。<br>（她背着父母，半夜偷偷玩游戏。） |
| 馬の耳に念仏（うまのみみにねんぶつ） | 对牛弹琴、无动于衷             | 比喻对不懂道理的人讲道理或对不理解的人讲深奥的东西，没有任何效果                  | 彼に説明しても**馬の耳に念仏**だ。<br>（跟他解释也是对牛弹琴。）             |
| 鼻が高い（はながたかい）           | 得意、自豪                   | 形容某人因自己的成就而感到自豪、得意洋洋，常用来表达自豪的语境                    | 彼は子供の成功を見て**鼻が高い**。<br>（看到孩子的成功，他感到非常自豪。）    |
| 耳にタコができる（みみにたこができる） | 听腻了、听烦了                | 表示因为同样的话或建议反复听了很多次，感到厌烦和腻味                               | 母に**耳にタコができる**ほど注意された。<br>（被母亲反复提醒，耳朵都听出茧子了。） |
| 手を貸す（てをかす）               | 伸出援手、帮忙                 | 表示在别人需要帮助时提供援助或协助，常用于表达提供帮助的行为                      | 引っ越しのときに**手を貸して**もらえますか？<br>（搬家时你能帮个忙吗？）       |
